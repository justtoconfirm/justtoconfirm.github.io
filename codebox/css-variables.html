CSS Variables

CSS Variables, sometimes referred to as Custom properties or cascading variables, are entities defined in CSS that contain specific values to be reused.

Advantages include:

  • Makes the code easier to read and more understandable.
  • Makes it much easier to change the colour values.
  • CSS variables have access to the DOM, which means they can be changed using JavaScript.

They are set using custom property notation and are accessed using the var() function.

Most browsers support CSS custom properties with the exception of Internet Explorer.

Declaring a custom property is done using a custom property name that begins with a double hyphen (--) and a property value that can be any CSS value.

    

--main-bg-color: #cceecc;

A common best practice is to define custom properties on the :root pseudo-class so that the styling can be applied globally across your HTML code.

    

:root {
    --main-bg-color: #cceecc;
}

You can use the custom property value by specifying it's name inside the var() function in place of a regular value. Here, we are applying the custom property value to set the background colour of the body element:

    

body {
    background-color: var(--main-bg-color);
}

Setting Fallbacks

The first argument of the var() function is the name of the custom property. We can apply a second argument to act as a fallback value which is used as a substitute when the custom property is invalid or not available. The var() function only accepts two parameters. If the second argument is also invalid, the fallback will fail.

    

body {
    background-color: var(--main-bg-color, #cdcdcd);    /* Fallback to #cdcdcd if --main-bg-color not defined */
}

Changing Variables

Sometimes we want the variables to change only in specific elements or sections of the page. Here we have an example of CSS variables being used to set the colour of HTML elements. Assume we want a different colour of blue for the button elements, we can re-declare the variable inside the button selector.

    

:root {
    --blue: #1e90ff;
    --white: #fff;
}

body {
    background-color: var(--blue);
}

button {
    --blue: #0000ff;            # Re-declare variable
    background-color: var(--white);
    color: var(--blue);         # This is now #0000ff;
    border: 1px solid var(--blue);
    padding: 5px;
}

You can use this same method to set local variables to specific elements if the code is to be used in one place.

    

button {
    --button-blue: #0000ff;
    color: var(--button-blue);
}
