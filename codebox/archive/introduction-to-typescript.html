Introduction to TypeScript

TypeScript is an open-source language built on JavaScript and developed by Microsoft. TypeScript is useful in the development of large applications and transcompiles to JavaScript. TypeScript first became popular with the release of Angular 2.

JavaScript code is valid in TypeScript code.

For TypeScript files, we use the file extension of .ts

Installing TypeScript

Before we start coding, we need to install TypeScript. We can use NPM (Node Package Manager) for this. Open your terminal and install the typescript package. This can be installed globally or as a devDependency (preference).

bash -- 85x24

    $ npm install typescript --save-dev

Once installed, a package.json file will be needed with a script to run TypeScript using the tsc command.

    

"scripts": {
    "typescript": "tsc"
},

If we have a TypeScript file named hello-world.ts, the package.json script may need to be updated to compile the file into JavaScript. This will generate a file named hello-world.js when executed.

    

"scripts": {
    "typescript": "tsc hello-world.ts"
},

Alternatively, if we want to compile all the TypeScript files inside any folder, we can use a wildcard.

    

"scripts": {
    "typescript": "tsc *.ts"
},
