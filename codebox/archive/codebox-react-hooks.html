React Hooks

React Hooks were introduced in React version 16.8.0 and allow us to use state and other React features without writing a class.

What is a Hook?

A Hook is a function that lets you "hook into" features of React. Hooks are prefixed with the keyword of use, followed by the name of the Hook.

An example of a common React Hook in use is useState which lets you add state to a function component.

Hooks can only be used inside a functional component, they cannot be used inside of a class component.

useState

Here we are going to use the useState hook to manage state within a React component. We begin by importing the useState hook.

	

import React, { useState } from 'react';

Next, we write our functional component. This component will create a counter.

	

import React, { useState } from 'react';

const Counter = () => {
	
	return (

	);
}

export default Counter;

Now that the component has been built, we need to apply the useState hook.

	

import React, { useState } from 'react';

const Counter = () => {

	// Hook
	const [ count, setCount ] = useState(4);
	
	return (

	);
}

export default Counter;
  • count - this is a variable and is our current state.
  • setCount - this is a function that allows us to update our state.

The default state, in this example, is set to 4. For a counter, you may want to set this to 0 as the default value.

We can now add two methods, one to increase and the other the decrease the count. For this we will call the setCount method to update the state. We pass in a new function, called prevCount into the setCount method to allow us to take the previous state value.

	

import React, { useState } from 'react';

const Counter = () => {

	// Hook
	const [ count, setCount ] = useState(4);

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (

	);
}

export default Counter;

We need to display the current count when the component renders. For this, we will use curly braces and reference the count variable.

	

import React, { useState } from 'react';

const Counter = () => {

	// Hook
	const [ count, setCount ] = useState(4);

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (
		<span>{ count }</span>
	);
}

export default Counter;

Finally, we can add two buttons that will call each function to increase, or decrease, the counter value.

	

import React, { useState } from 'react';

const Counter = () => {

	// Hook
	const [ count, setCount ] = useState(4);

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (
		<>
			<button onClick={decreaseCount}> - </button>

			<span>{ count }</span>

			<button onClick={increaseCount}> + </button>
		</>
	);
}

export default Counter;

We can make one final change to improve performance. We can update our hook to only run it once instead of everytime.

	

import React, { useState } from 'react';

const Counter = () => {

	// Hook - with improved performance
	const [ count, setCount ] = useState(() => {
		return 4;
	});

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (
		<>
			<button onClick={decreaseCount}> - </button>

			<span>{ count }</span>

			<button onClick={increaseCount}> + </button>
		</>
	);
}

export default Counter;

useEffect

Using the useEffect hook lets you perform side effects in functional components. The useEffect hook is similar to the componentDidMount and componentDidUpdate lifecycle methods.

We can update our earlier counter example to add the useEffect hook. This code will set the document title to a custom message that includes the counter value.

	

import React, { useState, useEffect } from 'react';

const Counter = () => {

	// Hook - with improved performance
	const [ count, setCount ] = useState(() => {
		return 4;
	});

	// The useEffect hook
	useEffect(() => {

		// Update the document title using the browser API
		document.title = `You clicked ${count} times.`;

	});

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (
		<>
			<button onClick={decreaseCount}> - </button>

			<span>{ count }</span>

			<button onClick={increaseCount}> + </button>
		</>
	);
}

export default Counter;
