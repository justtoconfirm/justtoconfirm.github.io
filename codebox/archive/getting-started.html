Building a Website: Getting started

With this guide we are going to build a website from the beginning to end.

Begin by installing a code editor to allow you to write your code. A few options are listed below:

  • Sublime Text
  • Atom
  • Visual Studio Code (VSCode)

Once installed, open your code editor and create a new HTML (HyperText Markup Language) file. All HTML files will have the file type of .html and we will save this in a folder somewhere on your machine where our website code will be located. In this example, our project folder will be called my-website.

Save the empty HTML file created as index.html inside the my-website folder.

The homepage of a website is often named index.html as most webservers recognise this file as the one to serve by default as the homepage.

Your file structure within your folder should look something like this:

    

my-website /
    index.html

Basic HTML structure

Within the index.html file, add the following HTML (HyperText Markup Language) code to build the basic template of our page. Once the code has been added save the file.

The file will now be a web page and can be opened in a web browser.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

</body>
</html>

When you open this page in a web browser no content will show, the page will be empty. Within the browser tab you will see the page title showing of My Webpage.

Any content we want to see on the page should be placed between the opening <body> and closing </body> HTML tags.

We will add the following heading to our HTML code in the index.html file.

    

<h1>Hello World</h1>

Once updated your code should look like the example below. Save the HTML file and refresh your web browser to see the heading appear.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

</body>
</html>

Next, under the heading add a paragraph as copy to the page. We will be using an opening <p> and closing </p> tag.

    

<p>This is an example paragraph.</p>

Our HTML code should now look like this:

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>

</body>
</html>

We can add multiple paragraphs if required.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>
    <p>This is yet another example paragraph.</p>

</body>
</html>

Adding Links

The HTML <a> tag, often called an anchor tag, is used to define a link (hyperlink). We can use links to navigate to other pages across our website or to external sites.

A link has an opening anchor tag <a> with a label for the link that appears on the page followed by a closing anchor tag </a>. This creates the link element.

We use a href attribute to specify the destination of the link. This is often a URL.

Here's an example link that will open the Hastings Direct homepage:

    

<a href="https://www.hastingsdirect.com">Hastings Direct</a>

As an example, add the link to your index.html page like so:

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>
    <p>This is yet another example paragraph.</p>

    <a href="https://www.hastingsdirect.com">Hastings Direct</a>

</body>
</html>

If we wanted to instead link to a page within our current website, we can also use the link element. For example, say we have a second HTML page in our website called about.html. This is how the file structure of our website would look:

    

my-website /
    index.html
    about.html

Within the href attribute of our link we simply add the name of the HTML page you want to open when you click the link.

    

<a href="about.html">About</a>

Meta tags

Meta tags define metadata about the HTML document and are always placed between the opening <head> and closing </head> tags of your HTML document. They are used to specify character set, page description, viewport settings, and much more.

We begin by specifying the character encoding for our HTML document. The charset attribute specifies this. This is usually UTF-8, which covers the most widely used characters and symbols.

    

<meta charset="utf-8">

Next, we add the viewport meta tag. This ensures that the browser has the instruction on how to control the dimensions and scale of the page and to ensure it is responsive on small screens/devices.

    

<meta name="viewport" content="width=device-width, initial-scale=1">

Adding the description meta tag will set the text that appears in SERPs (Search Engine Results Pages) when you search for your page. This is a consideration for SEO (Search Engine Optimisation).

    

<meta name="description" content="This is my example test home page to show my development skills">

Let's update the index.html code with the meta tags.

    

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <meta name="description" content="This is my example test home page to show my development skills">

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>
    <p>This is yet another example paragraph.</p>

    <a href="https://www.hastingsdirect.com">Hastings Direct</a>

</body>
</html>

Setting the language

The lang attribute helps to define the language of your HTML content. This is particularly useful for screen readers when thinking about accessibility as it defines how certain words or phrases are pronounced to users with visual impairments as well as setting languages for your audience if you are building a website for different geographical locations.

This attribute is added to the opening <html> tag.

By default, our page is English, so we will set the language to be English:

    

<html lang="en">

We could be more specific and set it to English (UK), if we prefer:

    

<html lang="en-GB">

Here are some of the common options:

    

<html lang="en">      // English
<html lang="es">      // Spanish
<html lang="fr">      // French
<html lang="de">      // German

Let's update our HTML code in the index.html file to set the language:

    

<!DOCTYPE html>
<html lang="en-GB">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <meta name="description" content="This is my example test home page to show my development skills">

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>
    <p>This is yet another example paragraph.</p>

    <a href="https://www.hastingsdirect.com">Hastings Direct</a>

</body>
</html>
