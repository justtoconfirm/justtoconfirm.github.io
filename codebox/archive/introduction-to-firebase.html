Introduction to Firebase

Firebase is a platform developed by Google for creating mobile and web applications.

https://firebase.google.com/

You will require a Google account, such as Gmail, to access Firebase.

Firebase Realtime Database

The Firebase Realtime Database is a cloud-hosted, NoSQL database. Data is stored as JSON (JavaScript Object Notation) and is synchronised in realtime to every connected client.

  • Create a Firebase project and in the Firebase console, click Add project.
  • Enter the name of your project.
  • Choose to setup Google Analytics for your project (optional).
  • Click Continue
  • Navigate to the Realtime Database section of the Firebase console and click the Create Database button.
  • Select Start in test mode to set the Security Rules for our database as we are creating a test database. This will set a trial period for when the test database access will be revoked (after 30 days) and an alert email will be sent once this occurs. Click Enable.

We now have a test realtime database created in Firebase. We now need to set the configuration for this database in our application using JavaScript.

Once the database has been created, select the Web option </> (next to the 'There are no apps in your project' label) in the Your apps panel. From here your code to embed into your HTML will be generated and should look like the example below:

Reading Data from the Realtime Database

    

var ref = firebase.database().ref();

ref.on("value", function(snapshot) {
    console.log(snapshot.val());
}, function (error) {
    console.log("Error: " + error.code);
});
