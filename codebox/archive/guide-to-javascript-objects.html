JavaScript Objects

Objects in JavaScript are used to group together variables and functions to act as a container for data values. The values are written as name:value pairs. The name:value pairs in a JavaScript object are called properties.

  • A variable is referred to as a property when in an object.
  • A function is referred to as a method when in an object.

It is common practice to declare (create) an object using the const keyword. Here we are creating an empty object called hotel. This is known as an object literal (also known as literal notation). This is often the easiest way to create objects in JavaScript.

			
const hotel = {}

We can now add properties to this object. Properties tell us about the object. All hotel objects will have the same properties, but the property values will differ from hotel to hotel.

			
const hotel = {
	name: "Quay",
	rooms: 150,
	booked: 90,
	gym: true,
	pool: true,
	roomType: ["Twin", "Double", "Suite"]
}

We can output the values of the properties within the object using dot notation like so:

			
objectName.propertyName

As an example, we can use dot notation to output the name of the hotel.

			
hotel.name

Our hotel object can be updated to add a method to display the number of rooms available in the hotel.

			
const hotel = {
	name: "Quay",
	rooms: 150,
	booked: 90,
	gym: true,
	pool: true,
	roomType: ["Twin", "Double", "Suite"],

	checkRoomAvailability() {
		return this.rooms - this.booked;
	}
}

To call the method, refence the object and then the name of the method itself.

			
hotel.checkRoomAvailability()

The this keyword is used to indicate that the rooms and booked properties of this object are being used.
