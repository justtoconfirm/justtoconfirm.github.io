JSON

JSON stands for JavaScript Object Notation. It works well with JavaScript as it's a superset of the JavaScript programming language.

JSON supports: Strings, Numbers, Boolean, null, Arrays and Objects.

  • Date Representation Format, like XML or YAML.
  • Commonly used for APIs and Configurations.
  • Lightweight and easy to Read/Write compared with something like XML.
  • Most programming languages allow you to work with JSON.

Create a file with the .json file extension.

An Array or an Object is usually at the top level of your JSON file. In the example below, we're using an object. The key and value pairs must use double-quotes.

    

{
    "key": "value",
    "key": "value"
}

Here's a further example:

    

{
    "name": "Dan",
    "favouriteNumber": 19,
    "isDeveloper": true,
    "hobbies": ["Football", "Programming", "Drawing"],
    "friends": [{
        "name": "Pearl",
        "favouriteNumber": 100,
        "isDeveloper": false
    }]
}

If we add JSON to a HTML page within a <script> tag:

    

<script>
let companies = 
[
    {
        "name": "Big Company",
        "employees": 2500,
        "ceo": "Darren",
        "rating": 4.5
    },
    {
        "name": "Small Company",
        "employees": 25,
        "ceo": "Dave",
        "rating": 5        
    }
]

console.log(companies);
</script>

Usually, when you work with JSON, such as with an API, the returned JSON will be as a string and not as an object - this is an emulation of this using backticks:

    

<script>
let companies = 
`[
    {
        "name": "Big Company",
        "employees": 2500,
        "ceo": "Darren",
        "rating": 4.5
    },
    {
        "name": "Small Company",
        "employees": 25,
        "ceo": "Dave",
        "rating": 5        
    }
]`

console.log(companies);
</script>

We can use JSON.parse() to convert this string to be a JSON object.

    

<script>
let companies = 
`[
    {
        "name": "Big Company",
        "employees": 2500,
        "ceo": "Darren",
        "rating": 4.5
    },
    {
        "name": "Small Company",
        "employees": 25,
        "ceo": "Dave",
        "rating": 5        
    }
]`

console.log(JSON.parse(companies));
</script>

If we wanted to get a specific value from our JSON object:

    

console.log(JSON.parse(companies)[1].name);     // Small Company
