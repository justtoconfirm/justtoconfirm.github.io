React

React is an open-source, front-end JavaScript library for building user interfaces and components.

Requirements

We will need to have the following installed on our machine to install and run React:

  • Node 14.0.0 or later
  • npm 5.6 or later

You can also install nvm (Node Version Manager) or nvm-windows to switch Node versions between projects.

Install

React can be installed using create-react-app. This will setup the development environment but will not handle backend logic or databases. This will only create the frontend build pipeline using Babel and Webpack, but we can use any backend we want.

To install React we will use npx, which is a package runner tool that comes with npm version 5.2 or later. This will install the current version of React.

In this example, our app is named my-app.

bash -- 85x24

	npx create-react-app my-app

Start the development server

Once React has been installed we can use the npm start command to run the local development server. This will run on port 3000 by default: http://localhost:3000/

bash -- 85x24

	cd my-app
	npm start

Hello World

Within the src/index.js file of the React app, simplify the code so we can display Hello World onto the page.

	

import React from 'react';
import ReactDOM from 'react-dom';

ReactDOM.render(
	<p>Hello World</p>,
	document.getElementById('root')
);

The attribute of root is the ID of the root element where the app will render in the public/index.html HTML document.

	

<div id="root"></div>

We call this a root DOM node inside our HTML because everything inside will be managed by the React DOM.

The className attribute

The class keyword is a reserved JavaScipt keyword. In React, we use className to add a class attribute to your elements and components.

	

<div className="foo"></div>

Embedding Expressions

We can use curly braces inside JSX to embed JavaScript expressions and data into our React application.

Here we declare a variable called name and then use it inside the curly braces to render the value:

	

import React from 'react';
import ReactDOM from 'react-dom';

const name = 'Dan';

ReactDOM.render(
	<p>Hello {name}</p>,
	document.getElementById('root')
);

As mentioned previously we can also put any valid JavaScript expression inside curly braces.

	

import React from 'react';
import ReactDOM from 'react-dom';

ReactDOM.render(
	<p>{2 + 2}</p>,
	document.getElementById('root')
);

Here is an example that renders the current year:

	

import React from 'react';
import ReactDOM from 'react-dom';

ReactDOM.render(
	<p>{new Date().getFullYear()}</p>,
	document.getElementById('root')
);

Components

Components let you split our User Interface (UI) into independent and reusable pieces. Components are like JavaScript functions. They accept inputs, called props (properties), and return React elements describing what should appear on the screen.

The simplest way to create a component is to write a JavaScript function.

Here we add a function as a component called Welcome and we can then pass in a value of a name into the props of our component:

	

import React from 'react';
import ReactDOM from 'react-dom';

const Welcome = (props) => {
	return (
		<p>Hello {props.name}</p>
	);
}

ReactDOM.render(
	<Welcome name="Dan" />,
	document.getElementById('root')
);

Composing Components

Components can refer to other components in their output. This lets us reuse components easily.

For example, we can create an App component that renders the Welcome component many times.

	

import React from 'react';
import ReactDOM from 'react-dom';

const Welcome = (props) => {
	return (
		<p>Hello {props.name}</p>
	);
}

const App = () => {
	return (
		<>
			<Welcome name="Dan" />
			<Welcome name="Pearl" />
			<Welcome name="Jack" />
		</>
	);
}

ReactDOM.render(
	<App />,
	document.getElementById('root')
);

Destructuring

We can apply destructuring to our React components for the props.

	

import React from 'react';
import ReactDOM from 'react-dom';

const Welcome = ({ name = '' }) => {

	return (
		<p>Hello {name}</p>
	);
}

const App = () => {
	return (
		<>
			<Welcome name="Dan" />
			<Welcome name="Pearl" />
			<Welcome name="Jack" />
		</>
	);
}

ReactDOM.render(
	<App />,
	document.getElementById('root')
);

We can also create an object to contain all of our props.

	

import React from 'react';
import ReactDOM from 'react-dom';

const Welcome = props => {

	const {
		name = '',
	} = props;

	return (
		<p>Hello {name}</p>
	);
}

const App = () => {
	return (
		<>
			<Welcome name="Dan" />
			<Welcome name="Pearl" />
			<Welcome name="Jack" />
		</>
	);
}

ReactDOM.render(
	<App />,
	document.getElementById('root')
);

Separating the Components

Now that we have a few components built, we can look to amend our file structure and seperate the components into individual files.

Inside the src directory, create a Welcome.js file that will contain the Welcome component.

	

import React from 'react';

const Welcome = props => {

	const {
		name = '',
	} = props;

	return (
		<p>Hello {name}</p>
	);
}

export default Welcome;

If not created already, inside the src directory, create an App.js file and update the code. We will import the Welcome component here.

	

import React from 'react';
import Welcome from './Welcome';

const App = () => {
	return (
		<>
			<Welcome name="Dan" />
			<Welcome name="Pearl" />
			<Welcome name="Jack" />
		</>
	);
}

export default App;

Our index.js file in the src directory will import the App component.

	

import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';

ReactDOM.render(
	<App />,
	document.getElementById('root')
);

The file structure of our app src directory will look like this:

	

my-app 
	|_src /
		|__App.js
		|__index.js
		|__Welcome.js

Conditional Rendering

In React, we can render components depending on the state of your application.

Consider these two components:

	

const UserGreeting = () => {
	
	return (
		<h1>Welcome back</h1>
	);

};

const GuestGreeting = () => {

	return (
		<h1>Please sign in</h1>
	);
	
};

We can create a Greeting component that displays either of these components depending on whether a user is logged in, or not.

	

import React from 'react';
import ReactDOM from 'react-dom';
import UserGreeting from './UserGreeting';
import GuestGreeting from './GuestGreeting';

const Greetings = props => {
	
	const isLoggedIn = props.isLoggedIn;

	if (isLoggedIn) {
		return <UserGreeting />;
	}

	return <GuestGreeting />;

}

ReactDOM.render(
	<Greeting isLoggedIn={false} />,
	document.getElementById('root')
);

We can also use ternary operators to conditionally render components based on data being passed in the props.

	

import React from 'react';

const Button = props => {
	
	const {
		type = "button",
		url = "",
		children,
		onClick
	} = props;

	const isAnchor = url && (url.includes('http') || url.startsWith('#') || url.startsWith('mailto'));

	const renderAsLink = () =>
		<a href={ url } role="button">{ children }</a>

	const renderAsButton = () =>
		<button {...{ type, onClick }}>{ children }</button>

	return (

		isAnchor ? renderAsLink() : renderAsButton()

	);
}

export default Button;

If we want to render the Button component as a button element:

	

<Button onClick={methodNameHere}>Click me</Button>

If we want to render the Button component as a link, we need to pass a value into a url prop:

	

<Button url="https://example.com">Click me</Button>
