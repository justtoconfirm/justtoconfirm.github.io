Go

Go, sometimes called Golang, is a statically typed, compiled programming language created by Google with a syntax similar to C.

Install Go

Begin by downloading the Go installer for your operating system.

Once installed, open your terminal and enter the command the confirm Go is installed:

bash -- 85x24

	go version

Getting Started

Once installed, open a command prompt and cd into your home directory. From here, create a hello directory for our first Go source code.

bash -- 85x24

	cd

	mkdir hello
	cd hello

Next, we need to add dependency tracking. This is defined by a go.mod file in our directory. We use the go mod init command to create this file. For the purpose of this example, we will just use example/hello.

bash -- 85x24

	go mod init example/hello

Create a file named hello.go within your directory and add the following code:

	

package main

import "fmt"

func main() {
	fmt.Println("Hello World")
}

Run the code from within the terminal to see the greeting.

bash -- 85x24

	go run .

External Packages

Visit pkg.go.dev and search for a "quote" package. Find the rsc.io/quote package.

Packages are published in modules.

In the Documentation section, under Index, note the list of functions we can call from our code. We will us the Go function.

In our code, import the rsc.io/quote package and add a call to it's Go function.

	

package main

import "fmt"
import "rsc.io/quote"

func main() {
	fmt.Println(quote.Go())
}

Run the following command to authenticate modules. This will add a go.sum file for use in authenticating the module we have just imported.

bash -- 85x24

	go mod tidy

Run the code to see the message generated by the function we're calling.

bash -- 85x24

	go run .

The file structure will look like this:

	

hello 
	|_go.mod
	|_go.sum
	hello.go
