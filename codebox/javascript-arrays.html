JavaScript Array

An Array in JavaScript is a type of global object that is used to store data. They are useful for storing multiple values in a single variable. Arrays can contain any datatype including numbers, strings and objects.

An empty array can be declared as with this example:

    

// Empty Array
const array = [];

This is known as an array literal.

Getting Items from an array

Each item in an array has an index number. The first item has the index of 0, the second item has the index of 1, and so on. If we want to select an item from an array, we use bracket notation and specify the index number.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams[0];       // Chelsea
teams[1];       // Liverpool
teams[2];       // Man Utd
teams[3];       // Millwall

Attempting to access an item that doesn't exist will return undefined.

    

teams[10];       // undefined

In order to access items in a nested array, you would need another index number to correspond to the inner array.

    

// Array of teams
const teams = [
    ['Chelsea', 'Liverpool', 'Man Utd', 'Arsenal'],
    ['Derby', 'Millwall', 'Sheffield Wednesday', 'Watford']
];

// Access both Arrays
teams[0][0];        // Chelsea
teams[0][1];        // Liverpool
teams[1][0];        // Derby
teams[1][1];        // Millwall

If you want to find out the index of a specific item in an array, we can use the indexOf() method. If an index number is not found, or a value does not exist, the console will return -1

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams.indexOf('Man Utd');       // 2

Adding & Removing items in an array

We can use the push() method to add a new item to the end of an array.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams.push('Leicester City');       // Chelsea, Liverpool, Man Utd, Millwall, Leicester City

The unshift() method will add an item to the beginning of an array.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams.unshift('Leicester City');    // Leicester City, Chelsea, Liverpool, Man Utd, Millwall

The first item from an array can be removed using the shift() method.

    

teams.shift();

Iteration

The forEach() method can be used to loop over each item contained within an array.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams.forEach(team => {
    console.log(team); 
});

We can also use a for loop which also uses the length property:

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];
// Declare variable so not to declare on each loop
let i;

for (i = 0; i < teams.length; i++) {
    console.log(teams[i]);
}

We can simplify our loop further when using ES6 syntax:

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];
// Declare variable so not to declare on each loop
let i;

// ES6 for loop syntax
for (i of teams) {
    console.log(i);
}

Getting the length of the array

The length property can be used to count the total number of items in an array.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

console.log(teams.length);      // 4

We can also use the length property to find the last item in an array.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];
// The last item in the Array
const lastItem = teams.length -1;

teams[lastItem];        // Millwall

Changing items in an Array

Items within an array can be changed by specifying the index of the item and assigning a new value. In this example, we will change the value of Liverpool to be Arsenal.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

teams[1] = 'Arsenal';

Array Destructuring

Array destructuring allows you to create new variables using an array item as a value. Consider this example:

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

Arrays in JavaScript preserve their order, so the first item will always have the index of 0, the second will have the index of 1, and so on. Knowing this, you can create variables from items in the array but this is a manual process and can take up space in your code.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

const teamOne = teams[0];       // Chelsea
const teamTwo = teams[1];       // Liverpool
const teamThree = teams[2];     // Man Utd
const teamFour = teams[4];      // Millwall

With array destructuring, you can unpack the values from the array in order and assign them to their own variables.

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

const [teamOne, teamTwo, teamThree, teamFour] = teams;

Values can be skipped by leaving the destructuring syntax blank between items:

    

// Array of teams
const teams = ['Chelsea', 'Liverpool', 'Man Utd', 'Millwall'];

const [teamOne, , teamThree, teamFour] = teams;

These variables can then be logged:

    

console.log(teamOne);       // Chelsea
console.log(teamTwo);       // Liverpool
console.log(teamThree);     // Man Utd
console.log(teamFour);      // Millwall

Nested arrays can also be destructed. First, create a nested array:

    

// Nested Array
const nestedArray = [1, 2, [3, 4], 5];

We can then destructure that array and log the new variables:

    

// Nested Array
const nestedArray = [1, 2, [3, 4], 5];

// Destructure the Array and nested items
const [one, two, [three, four], five] = nestedArray;

// Log the output
console.log(one, two, three, four, five);       // 1 2 3 4 5
