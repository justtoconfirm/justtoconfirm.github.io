Browsers

A Web Browser (commonly referred to as a Browser) is a software application for accessing the World Wide Web. Examples of Web Browsers include:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge
  • Microsoft Internet Explorer (IE)
  • Opera
  • Brave

Are Browsers so different?

Despite a variety of applications, all browsers have the same goal: to render web pages. They achieve this with a rendering engine and many browsers use the same rendering engine.

  • Safari / Chrome (iOS) - Webkit
  • Chrome / Edge (Chromium) / Opera / Brave / Vivaldi - uses Blink (Chromium-based), a fork of Webkit
  • Firefox - Gecko
  • Internet Explorer - Trident
  • Edge (Legacy) - EdgeHTML, an update of Trident used in Internet Explorer (IE). This version of Edge is to no longer be supported.

Google Chrome

  • Released: 2008
  • Rendering Engine: Blink
  • JavaScript Virtual Machine (JVM): V8 JavaScript engine

Google Chrome, also referred to simply as Chrome, is a cross-platform web browser developed by Google and first released in 2008. Chrome is the default browser built into the Android OS and is the main component of Chrome OS, where is serves as the platform for web applications.

Most of Chrome's source code comes from Google's open-source project Chromium. WebKit was originally used as the rendering engine but this has now been forked to create the Blink engine. All Chrome versions, except the iOS version, now use Blink as the rendering engine for the browser.

Microsoft Edge

  • Released: 2015
  • Rendering Engine: EdgeHTML (now replaced by Chromium-based Blink)
  • JavaScript Virtual Machine (JVM): Chakra (now replaced by the V8 JavaScript engine)

Formally codenamed Project Spartan/Anaheim, Microsoft Edge is the replacement browser for Internet Explorer (IE).

Edge is the default web browser on Windows 10, Windows 10 Mobile and Xbox One consoles.

The non-Chromium version of Edge was initially built using Microsoft's own proprietary browser engine, EdgeHTML and their Chakra JavaScript engine (a version referred to as Microsoft Edge Legacy). It was announced on January 15, 2020 that a new Chromium-based version of Edge would replace the non-Chromium version. In June 2020, Microsoft began to rollout the new version of Edge for during an update for Windows 7, 8.1 and 10 versions. The current version of Edge now uses the same Chromium-based Blink rendering engine and V8 JavaScript engine as found in Google Chrome.

EdgeHTML was the proprietary layout engine originally developed for Edge. It was a fork of the Trident engine but with the legacy Internet Explorer (IE) code removed and rewritten to support modern web standards. EdgeHTML was written in C++

  • March 9, 2021 - Microsoft to drop support for existing legacy versions of Edge (non-Chromium). After this date, the legacy version of Edge will no longer receive security updates. Migration of the current Edge (Chromium) advised.

Microsoft has been moving existing Windows 10 users over to the new Chromium-based Edge with Windows updates.

Microsoft Internet Explorer (IE)

Microsoft Internet Explorer (IE) is a web browser developed by Microsoft and released in 1995. It has since been replaced by Edge as the default web browser for Microsoft Windows.

  • Released: August 16, 1995
  • Rendering Engine: Trident
  • August 17, 2021 - Internet Explorer 11 (IE11) will no longer be supported by Microsoft for online services like Office 365, OneDrive and Outlook.
  • November 30, 2021 - Internet Explorer 11 (IE11) will no longer be supported in Microsoft Teams and any further applications.

Microsoft is expected to drop support for Internet Explorer 11 within Windows 10.

The Chromium-based version of Microsoft Edge will continue to let businesses access websites, applications and tools built for Internet Explorer using the Internet Explorer legacy mode ("IE Mode"). This mode uses the IE rendering engine allowing IE to be built directly into Edge for this purpose. A small 'IE logo' will appear inside the address bar when the mode is active.

The deadline for support to end for IE can be viewed in real-time on the death-to-ie11.com website.

Mozilla Firefox

  • Released: September 23, 2002
  • Rendering Engine: Gecko

Mozilla Firefox, or simply Firefox, is an open-source web browser developed by the Mozilla Foundation.
