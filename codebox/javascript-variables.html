JavaScript Variables

A variable is a "named storage" for storing data. We can use JavaScript variables. To create a variable in JavaScript, use the let keyword. The statement below declares (creates) a variable with the name message.

JavaScript is an untyped language meaning that a variable can hold any data type value. You do not need to declare the data type as JavaScript handles this automatically during the execution of the code.

In order to use a variable you must give it a name. This is sometimes called the identifier. If a variable name is more than one word, it's usually written in camelCase.

    

let message;
let firstName;      // Identifier (name) in camelCase   

var message;        // Legacy var keyword

The keyword of var was used in JavaScript to declare variables, but this has since been replaced by the use of let and/or const.

A variable should be declared only once. A repeated declaration of the same variable will cause a SyntaxError.

Until a value has been assigned to a variable, it's value is undefined (an empty container). You assign values to variables. We can assign a value to a variable by using the assignment operator (=), like so:

    

let message = 'Hello World';

The data within the variable can be accessed/returned using the variable name.

    

alert(message);         // Hello World
