Introduction to WordPress

WordPress (WP) is an open-source Content Management System (CMS) written in PHP and using a MySQL or MariaDB database.

How to setup WordPress

WordPress can be downloaded from the WordPress.org website. The latest version of WordPress is version 5.8. It is recommended to never edit the core code of WordPress.

The recommended requirements:

  • Apache or Nginx web server
  • PHP version 7.4 or greater
  • MySQL version 5.6 or greater / MariaDB version 10.1 or greater

Install the web server

To setup the Apache or Nginx web server locally, install one of these recommended tools:

  • MAMP PRO (Mac)
  • MAMP PRO (Windows)
  • WampServer
  • XAMPP

WordPress houses lots of different types of content that are divided into something called Post Types. By default WordPress stores all post types in the database under the wp_posts table.

Posts

Posts are entries listed in reverse chronological order on your site. Think of them as articles, or blog posts, that can be displayed on your site.

A post is an example of a Post Type in WordPress. This will be stored in the wp_posts table within the database.

To create your first post, log in to your WordPress dashboard and navigate to Posts/Add New.

When you create a post, you can assign it to a category and add tags, so your visitors can browse content by topic rather than chronologically by date.

  • Categories - are like a table of contents for your WordPress site. For instance, if you sell merchandise, one of your categories maybe "Shirts".
  • Tags - are granular terms that are specific to a piece of content. For instance, if you want to publish an announcement about a new Shirt, you may use tags like "Organic Cotton" or "Sports" as descriptors.

Pages

Pages are used for static content that is not affected by date. For example, an About page, Contact page, or Home page, are examples of pages. We may also have a Blog page used to display posts. Pages are typically listed in your site's navigation.

A page is an example of a Post Type in WordPress. This will be stored in the wp_posts table within the database.

To create your first page, log in to your WordPress dashboard and navigate to Pages/Add New.

Pages do not use categories or tags, unlike posts. Instead, pages can be organised into parent and child page groups to group them hierarchically. For instance, you may have a parent page called "Apparel" which can then be split into "Women's Apparal" and "Men's Apparal".

Plugins

WordPress plugins are PHP scripts that extend the functionality or add new features to WordPress. Any plugin installed will be placed within the wp-content/plugins/ directory of your WordPress installation.

It's possible to install plugins manually by extracting and copying them into the plugins directory. Doing so will make them available for activation within the WordPress admin dashboard.

Advanced Custom Fields (ACF)

Advanced Custom Fields (ACF) is a WordPress plugin that allows you to add extra content fields, known as Custom Fields, to the edit screens within the admin dashboard of WordPress.

This plugin does not come pre-installed with WordPress so needs to be installed to be used. The plugin will be installed in the wp-content/plugins/ directory of your WordPress installation.

Once installed, we will see a Custom Fields tab in the sidebar of the WordPress admin dashboard.

Themes

WordPress sites can be styled using themes. We have the option to install an existing theme or we can choose to create our own custom theme. A list of available themes available to install can be found on the WordPress.org website.

Any theme installed will be inside the wp-content/themes/ directory of your WordPress installation. WordPress 5.8 comes pre-installed with the Twenty Twenty-One theme by default. This theme can be changed.

A WordPress theme needs two files to exist - style.css and index.php. Some themes will also have a functions.php file. This is known as a theme functions file and acts like a plugin for the WordPress site that's automatically activated within the current theme.

To view the templates from within the WordPress admin dashboard, open the Appearance/Themes tab from within the dashboard sidebar.
