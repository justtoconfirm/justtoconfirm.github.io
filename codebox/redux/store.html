Store

A store is a state container which holds the application's state. Redux can have only a single store in your application. Whenever a store is created in Redux, you need to specify the reducer.

Setup React

Begin by using the create-react-app tool to setup React. Here, we setup React as a TypeScript application.

bash -- 85x24
    
	npx create-react-app APP-NAME-HERE --template typescript

Install Redux Toolkit & react-redux

Once React is setup, we can install Redux using the Redux Toolkit (RTK). This will also install react-redux.

bash -- 85x24
    
	npm install @reduxjs/toolkit react-redux

Create the Store (Redux Toolkit)

Create a file named src/app/store.js. Import the configureStore API from Redux Toolkit. Here we then have an empty Redux store that we can then export.

	

import { configureStore } from '@reduxjs/toolkit';

export const store = configureStore({
	reducer: {},
});

In TypeScript, this will be named src/app/store.tsx:

	

import { configureStore } from '@reduxjs/toolkit';

export const store = configureStore({
	reducer: {},
})

// Infer the `RootState` and `AppDispatch` types from the store itself
export type RootState = ReturnType<typeof store.getState>

// Inferred type: {posts: PostsState, comments: CommentsState, users: UsersState}
export type AppDispatch = typeof store.dispatch

This creates a Redux store and automatically configures the Redux DevTools extension so you can inspect the store during development.

Add Slice to the Store

To add a slice to the store, we just need to define a field inside the reducer parameter, we then tell the store to use this slice reducer function to handle all updates to that state.

	

import { configureStore } from '@reduxjs/toolkit';
import exampleReducer from '../slices/exampleSlice';

export const store = configureStore({
	reducer: {
		example: exampleReducer,
	},
});

Connect Store to React

Once the store is created, we can make it available to our React components by putting a react-redux <Provider> around our application in the src/index.js file (or src/index.tsx if TypeScript). 

	

import React from 'react';
import ReactDOM from 'react-dom';
import { store } from './app/store';
import { Provider } from 'react-redux';

ReactDOM.render(
	<Provider store={store}>
		<App />
	</Provider>,
	document.getElementById('root')
);

File structure

This is the basic file structure within the src directory:

	

src /
	|_app /
		|_store.js
	|_slices /
		|_exampleSlice.js
	|_index.js
	|_index.css
