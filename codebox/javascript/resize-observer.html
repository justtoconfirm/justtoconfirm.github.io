Resize Observer API

The Resize Observer API allows code to monitor an element for changes to its size.

Resize Observer

The Resize Observer is used to observe individual elements on the page using JavaScript. CSS media queries are used to target the entire page.

This is the HTML code:

	

<!DOCTYPE>
<html lang="en">
<head>
	<meta charset="utf-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">

	<title>Resize Observer</title>

	<link rel="stylesheet" href="style.css">

	<script src="script.js" defer></script>
</head>
<body>

	<main role="main">
		<div class="container">
			<div class="box"></div>
		</div>
	</main>

</body>
</html>

Save the code below as a JavaScript file named script.js.

	

const box = document.querySelector('.box');
const container = document.querySelector('.container');

// Return an array called entries for all the things we want to observe
const observer = new ResizeObserver((entries) => {

	// Get the first element from the array, the box
	const boxElement = entries[0];
	// If box is less than 150 px
	const isSmall = boxElement.contentRect.width < 150;

	// Change the colour of the box if size is below 150 px
	boxElement.target.style.backgroundColour = isSmall ? "blue" : "red";
	
	// This will output in the log each time the elements change size
	console.log(entries);

});

// Observe a single item called box
observer.observe(box);
// Observe a single item called container
observer.observe(container);

Browser Support

  • Chrome - 64 and greater
  • Edge - 79 and greater
  • Firefox - 69 and greater
  • Internet Explorer (IE) - No
  • Safari - 13.1 and greater
