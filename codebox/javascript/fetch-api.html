Fetch API

The Fetch API provides a JavaScript interface for accessing and manipulating parts of the HTTP pipeline, such as requests and responses.

Fetch API

The Fetch API provides functionality that was previously achieved using the XMLHttpRequest (XHR) object. Fetch API provides a better alternative that can easily used by other technologies such as Service Workers and a logical place to define other HTTP-related concepts such as CORS (Cross-Origin Resource Sharing).

A basic fetch requst is simple to set up:

	

fetch('https://api.coindesk.com/v1/bpi/currentprice.json')
	.then(response => response.json())		// Convert response to JSON object
	.then(data => console.log(data));

Here we are fetching JSON code showing the current Bitcoin price and printing it to the console. The Fetch API takes one argument - the path to the resource you want to fetch - this does not directly return the JSON response, but instead returns a promise.

We use the json() method to extract the JSON body content from the Response object.
