Objects

JavaScript Objects are containers for named values called properties. Objects can also have methods.

Creating Objects

Objects can be created using literal notation:

	

const hotel = {
	name: 'Quay',
	rooms: 40,
	booked: 25,
	
	checkAvailability: function() {
		return this.rooms - this.booked;
	}
}

We can then return the properties and/or call the methods within the object:

	

hotel.name

hotel.checkAvailability();

Sometimes you may want to create multiple objects to represent similar things. Object constructors can use a function as a template for creating objects.

	

const hotel = (name, rooms, booked) => {
	this.name = name;
	this.rooms = rooms;
	this.booked = booked;
	
	this.checkAvailability = function() {
		return this.rooms - this.booked;
	}
}

We can then use the new keyword to create multiple hotel objects, we can just get the object and pass in the data:

	

const quayHotel = new Hotel('Quay', 40, 25);
const parkHotel = new Hotel('Park', 120, 71);

We can then output the properties and/or methods of each object using a reference to the variable name.

	

quayHotel.name;

quayHotel.checkAvailability();

Object.values() method

The Object.values() method returns an array of a given object's property values:

	

const hotel = {
	name: 'Quay',
	rooms: 40,
	booked: 25
}

Object.values(hotel);		// (3) ['Quay', 40, 25]

This method was introduced in ECMAScript 8 (ES8) and works similar to Object.keys(). It returns the value of all properties in an object as an array. We can then loop through these values in the array by usin the forEach() method.

	

Object.values(hotel).forEach(val => console.log(val));

Object.keys() method

The Object.keys() method returns an array of a given object's property names (keys):

	

const hotel = {
	name: 'Quay',
	rooms: 40,
	booked: 25
}

Object.keys(hotel);		// (3) ['name', 'rooms', 'booked']
