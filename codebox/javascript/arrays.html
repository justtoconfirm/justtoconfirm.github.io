Arrays

An array os a variabe that can hold more than one value.

Creating an Array

Here we create an empty array. To create an array, we use square brackets []. This will create an empty array.

	

const a = [];

Adding items to the array is really simple. Here we have an array with four elements:

	

const a = ["A", "B", "C", "D"];

Every element in an array has an index. The first element, A, has an index of 0 and the final element, D, has an index of 3.

Accessing Elements in an Array

Now we have created an array, to access an element within the array, we can reference the name of the array followed by square brackets that will contain the index of the element we want to return. For example, if we want to return B, we will use index 1.

	

const a = ["A", "B", "C", "D"];

console.log(a[1]);

We will return undefined if we attempt to get an element from our array that does not exist.

If we want to get the number of total elements in our array, the length, we can use the length property. This will return 4 as we have 4 elements.

	

const a = ["A", "B", "C", "D"];

console.log(a.length);

Changing Elements in an Array

We can change elements in an array by using the index. For example, we can change "C" to be the number 22, like so:

	

const a = ["A", "B", "C", "D"];

a[2] = 22;

console.log(a);		// (4) ["A", "B", 22, "D"]

Adding Elements to an Array

To add a new element to an array, we can use the push() method. This code example, below, will add "E" to our array.

	

const a = ["A", "B", "C", "D"];

a.push("E");

We can also push another array into our array if we choose.

	

const a = ["A", "B", "C", "D"];

a.push([1, 2, 3]);

This added array would have the index of 4.

Nested Arrays

This is similar to a nested array:

	

const a = [
	["A", "B", "C", "D"],
	[1, 2, 3],
];

To access the second element of the first array - "B" - we can use the following code:

	

const a = [
	["A", "B", "C", "D"],
	[1, 2, 3],
];

console.log(a[0][1]);

The first index - [0] - is the first array, the second index - [1] - is the second element of that array.
