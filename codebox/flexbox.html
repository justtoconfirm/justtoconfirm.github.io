Flexbox

The CSS Flexible Box Layout, or Flexbox, was introduced in 2009 and is a one-dimensional layout model. When we describe flexbox as being one-dimensional, we are describing Flexbox as dealing with layout in one dimension at a time - either as a row, or as a column. This is in contrast to the CSS Grid, which controls both rows and columns together.

We begin by setting the display property on your container element to flex.

    

.flex {
    display: flex;
}

This is our HTML code:

    

<div class="flex">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
</div>

The direct child elements of a flex container automatically becomes a flexible (flex) item.

By default, the flex display value sets the direction to be a row.

Setting Columns & Rows

Flexbox has a main axis defined by the flex-direction CSS property. This property has four possible values:

  • row
  • row-reverse
  • column
  • column-reverse

Should you set the value as row, the axis will run along the row in the inline direction. Choosing column will set the axis to run from top to bottom in the block direction.

    

.flex {
    display: flex;
    flex-direction: row;
}

Flex Basis

The flex-basis is what defines the size of each item. The initial value of this property is auto. If the items do not have a size, the content's size is used as the flex-basis. By default the items in a row will only take as much space as they need to display their contents.

This property specifies the initial length of a flex item.

If we were to set the value of flex-basis to 100px, the items would have a width of 100px. You will need to set the flex-basis on the items within the container for this to take effect.

    

.flex {
    display: flex;
}

.flex__item {
    flex-basis: 100px;
}

A Responsive Flexbox Grid

We can use the flex-basis property to create a simplified responsive grid using Flexbox.

    

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

.column {
    flex-basis: 100%;
}

@media screen and (min-width: 800px) {
    .column {
        flex: 1;
    }
}

We can change the widths of our columns to make the grid more flexible using the flex property.

    

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

.column {
    flex-basis: 100%;
}

@media screen and (min-width: 800px) {
    .column {
        flex: 1;
    }

    .column-25 {
        flex: 2.5;
    }

    .column-55 {
        flex: 5.5;
    }

    .column-20 {
        flex: 2;
    }
}

The grid should be wrapped inside a container so the content doesn't stretch beyond 1200px on large screens.

    

.container {
    max-width: 1200px;
    margin: 0 auto;
}

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

.column {
    flex-basis: 100%;
}

@media screen and (min-width: 800px) {
    .column {
        flex: 1;
    }

    .column-25 {
        flex: 2.5;
    }

    .column-55 {
        flex: 5.5;
    }

    .column-20 {
        flex: 2;
    }
}

Now if we want to create a row with three columns, we can write our HTML like so:

    

<div class="container">
    <div class="row">
        
        <div class="column">
            One
        </div>
        <div class="column">
            Two
        </div>
        <div class="column">
            Three
        </div>

    </div>
</div>

Each column will have a 100% width on mobile having used the flex-basis property and will wrap.

Wrapping

While Flexbox is one dimensional, it is possible to cause our flex items to wrap onto multiple lines.

To apply wrapping behavior, add the flex-wrap property with the value of wrap. The initial value is set to nowrap. This will instead shrink to fit the container because they are using the initial flexbox values that allow items to shrink, but will cause an overflow if items are not able to shrink.

Now, if your items become too large to display on a single row, they will wrap onto another line.

    

.flex {
    display: flex;
    flex-wrap: wrap;
}

.flex__item {
    flex-basis: 250px;
}

The flex-flow shorthand

We can combine both the flex-direction and flex-wrap properties into the shorthand flex-flow property.

The first value specified is the flex-direction and the second is the flex-wrap.

    

.flex {
    display: flex;
    flex-flow: row wrap;
}

Justify Content

The justify-content property is used to align the items on the main axis. The initial value is flex-start. This will set all the items on the left at the start edge of the container.

    

.flex {
    display: flex;
    justify-content: flex-start;
}

Centering

We can use the justify-content and align-items properties with value of center to perfectly centre a flex item both horizontally and vertically.

    

.flex {
    display: flex;
    height: 300px;
    justify-content: center;
    align-items: center;
}

The flex-grow property

The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items.

The default value is set to 0 and a number should be applied to the property.

One
Two
Three

Here's the CSS:

    

.flex {
    display: flex;
    align-items: stretch;
}

.flex__item--grow {
    flex-grow: 8;
}

If we set the flex-grow property to a value of 1 for each flex item, each item will have an equal size inside the flex container.

    

.flex {
    display: flex;
}

.flex__item {
    flex-grow: 1;
}

Gap

We can use the gap property to apply spacing between the items in our Flexbox container.

In this example, we can equally space out our items in the row axis with no margin or padding needed.

    

.flex {
    display: inline-flex;
    gap: 12px;
}

Setting the display property to inline-flex does not make the flex items display inline, it makes the flex container display inline.

You may want to add gap spacing using em or rem units instead of px.

Building a Navigation bar

We can use Flexbox to build a navigation bar for a website.

Often, the navigation is placed inside of the HTML5 <header> element and within a <nav> element.

    



    <a href="/">
        <img src="logo.svg" height="50" width="100" alt="">
    </a>

    <nav role="navigation">
        <ul>
            <li><a href="/">Home</a></li>
            <li><a href="/about.html">About</a></li>
            <li><a href="/blog.html">Blog</a></li>
        </ul>
    </nav>
</header>

Or CSS will look something like this:

    

header {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

We first apply flex using the display property, then we add space between the elements inside the <header> (the <a> and the <nav> elements), so that they are positioned at either end of the row, finally we align the elements vertically-centered using the align-items property.

On the unordered list containing our navigation links, we can use this CSS:

    

ul {
    display: flex;
    flex-wrap: wrap;
    list-style-type: none;
    gap: 4rem;
}
