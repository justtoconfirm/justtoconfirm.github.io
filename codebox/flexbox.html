Flexbox

The CSS Flexible Box Layout, or Flexbox as it's known, was introduced in 2009 and is a CSS3 web layout model that allows responsive elements within a container to be automatically arraged depending upon screen size.

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit smaller spaces.

Flexbox is supported across all modern browsers. IE11 has partial support.

There are three versions of the Flexbox spec:

    

.container {
    display: flex;      // This is the one to use
    display: flexbox;   // This was a draft spec implemented in IE10, avoid if possible
    display: box;       // This is no longer used
}

Specifying the layout to use Flexbox

To begin, we need to first select the elements that are to be laid out as flexible boxes. To do this, we set the display property in our CSS on the parent element. This causes the element to become a flex container and its children to become flex items.

Summary: The flex container holds the flex items.

    

.container {
    display: flex;
}

With this one line of CSS code, every item inside the element turns into a flex item. The main direction that our flex items take in our flex container is a row.

You can set the flex-direction here to set the direction as a row, but it's not necessary:

    

.container {
    display: flex;
    flex-direction: row;
}

The HTML for this example may look something like this:

    

<div class="container">
    <div class="container__item">One</div>
    <div class="container__item">Two</div>
    <div class="container__item">Three</div>
</div>

Preferred Size

The preferred size of a container/box is the size defined by a width or a height.

    

.container__item {
    width: 150px;
}

We can also use the logical alias in the inline direction.

    

.container__item {
    inline-size: 150px;

The flex-wrap property

The flex-wrap CSS property sets whether flex items are forced onto one line or wrap onto multiple lines.

    

.container {
    flex-wrap: wrap;
}

We can see here that the flex items wrap within the container if they cannot fit if we set the flex-wrap property value to be wrap.

Responsive Flexbox

We can use media queries to create different layout for different screen sizes and devices.

This code will display the items in a row on desktop (above 800px width) and stack the items as columns on mobile screens (below 800px) using a media query and the flex-direction property.

    

.container {
    display: flex
    flex-direction: row;
}

@media (max-width: 800px) {
    .container {
        flex-direction: column;
    }
}

The flex property

The flex property is a shorthand property for the flex-grow, flex-shrink, and flex-basis properties. This property sets the flexible length on the flex items within the container.

The second and third parameters of flex-shrink and flex-basis, are optional.

  • flex-grow
  • flex-shrink (optional)
  • flex-basis (optional)
    

.container__item {
    flex: flex-grow | flex-shrink | flex-basis
}

/* OR */

.container__item {
    flex: max | min | ideal size
}

Using the below CSS code will set all flexible items to be the same length, regardless of content.

    

.container__item {
    flex: 1;
}

flex-grow

We are essentially setting the flex-grow property here using the shorthand flex property set to 1, meaning every item will set an equal size inside the flex container. This is the same as writing:

    

.container__item {
    flex-grow: 1;           // is the same as flex: 1;
}

If we were to give one of the child items a value of 2, that item would take up twice as much space as the others in the flex container.

    

.container__item:nth-of-type(1),
.container__item:nth-of-type(2) { 
    flex-grow: 1;           // is the same as flex: 1;
}

.container__item:nth-of-type(3) { 
    flex-grow: 2;           // is the same as flex: 2;
}

We can see that the third item is much larger in length than the first and second items.

flex-basis

The flex-basis property specifies the initial size of the flex item before any avialable space is distributed. When omitted from the flex shorthand property, its value is zero.

The default value for flex-basis is auto.

flex-basis is the third (optional) parameter in the flex shorthand property.

Here we're setting the flex-basis of the second item to set the initial size to be 50%.

    

.container__item:nth-of-type(2) { 
    flex-basis: 50%;    
}

Spacing

Here we will create a container with a number of items. The items will be spaced out with no margin or padding on the very end of the row:

    

<div class="flex-gap">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
</div>

.flex-gap {
    display: inline-flex;

Building a Flexbox Grid

We're going to now build a grid using Flexbox in the vein of Bootstrap, Foundation and Skeleton. With only two classes of row and column, we can create an infinite equally spaced grid.

    

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

.column {
    flex-basis: 100%;
}

@media screen and (min-width: 800px) {
    .column {
        flex: 1;
    }
}

In a framework like Bootstrap, the grid is based on 12 columns. With flexbox, this can be simplified within our HTML code. This example will provide us with one 100% wide column.

    

<div class="row">
    <div class="column">
        <!-- 100% width -->
    </div>
</div>

The width of each column is 100% divided by the number of columns within the flex container row.

    

<div class="row">
    <div class="column">
        <!-- 50% width -->
    </div>
    <div class="column">
        <!-- 50% width -->
    </div>
</div>

This is another example showing equally dividing columns:

    

<div class="row">
    <div class="column">
        <!-- 33.3% width -->
    </div>
    <div class="column">
        <!-- 33.3% width -->
    </div>
    <div class="column">
        <!-- 33.3% width -->
    </div>
</div>

We can change the widths of our columns to make the grid more flexible. We can use the flex property inside of a media query to create columns that are unique widths.

    

@media screen and (min-width: 800px) {
    .column_25 {
        flex: 2.5;
    }

    .column_55 {
        flex: 5.5;
    }

    .column_20 {
        flex: 2;
    }
}

We are currently using a single breakpoint to keep things simple for mobile/tablet (small screens) and desktop/laptop (large screens), however this can be extended further if needed. Using min-width media queries is optimal for mobile first layout designs.

Each column will have a 100% width on mobile having used the flex-basis property and will wrap down to the next column. On desktop, all columns will be contained to a single line. The flex-basis property specifies the initial main size of a flex item inside a flex container.

Our HTML may look something like this:

    

<div class="row">
    <div class="column column_25">
        <!-- 25% width -->
    </div>
    <div class="column column_55">
        <!-- 55% width -->
    </div>
    <div class="column column_20">
        <!-- 20% width -->
    </div>
</div>

We should wrap the grid inside of a container so your content doesn't stretch beyond 1200px on large screens.

    

.container {
    max-width: 1200px;
    margin: 0 auto;
}

The HTML will look like this:

    

<div class="container">
    <!-- rows and columns -->
</div>
