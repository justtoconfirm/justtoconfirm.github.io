Flexbox

The CSS Flexible Box Layout, or Flexbox as it's known, was introduced in 2009 and is a CSS3 web layout model that allows responsive elements within a container to be automatically arraged depending upon screen size.

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit smaller spaces.

Flexbox is supported across all modern browsers. IE11 has partial support.

There are three versions of the Flexbox spec:

    

.container {
    display: flex;      // This is the one to use
    display: flexbox;   // This was a draft spec implemented in IE10, avoid if possible
    display: box;       // This is no longer used
}

Specifying the layout to use Flexbox

To begin, we need to first select the elements that are to be laid out as flexible boxes. To do this, we set the display property in our CSS on the parent element. This causes the element to become a flex container and its children to become flex items.

Summary: The flex container holds the flex items.

    

.container {
    display: flex;
}

With this one line of CSS code, every item inside the element turns into a flex item. The main direction that our flex items take in our flex container is a row.

You can set the flex-direction here to set the direction as a row, but it's not necessary:

    

.container {
    display: flex;
    flex-direction: row;
}

The HTML for this example may look something like this:

    

<div class="container">
    <div class="container__item">One</div>
    <div class="container__item">Two</div>
    <div class="container__item">Three</div>
</div>

Preferred Size

The preferred size of a container/box is the size defined by a width or a height.

    

.container__item {
    width: 150px;
}

We can also use the logical alias in the inline direction.

    

.container__item {
    inline-size: 150px;
}

The flex-wrap property

The flex-wrap CSS property sets whether flex items are forced onto one line or wrap onto multiple lines.

    

.container {
    flex-wrap: wrap;
}

We can see here that the flex items wrap within the container if they cannot fit if we set the flex-wrap property value to be wrap.
