CSS Grid

A grid is a set of horizontal and vertical lines that define columns and rows. Elements can be placed onto the grid within these columns and rows to create the page layout.

CSS Grid Layout introduces a two-dimensional grid system to CSS.

Browser support for CSS Grid is good across all modern browsers, with only Internet Explorer having partial support.

Creating the Grid container

We create a grid container using the display property on an element. As soon as we do this, the children of the element become grid items.

    

.container {
    display: grid;
}

Our HTML will look something like this:

    

<div class="container">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
</div>

All the direct children are now grid items. The grid has created a single column for the grid items.

Grid Tracks

We define rows and columns on our grid with the grid-template-columns and grid-template-rows properties. These define grid tracks, a space between any two lines on the grid. The grid tracks can be defined using any length unit.

CSS Grid introduced the fr unit to represent a fraction of the available space in a grid container.

    

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}

This can also be written using the repeat() notation, to repeat all or a section of the track listing.

    

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Here we are setting the columns so that the three columns have equal space.
