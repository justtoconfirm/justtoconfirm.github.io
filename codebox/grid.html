CSS Grid

A grid is a set of horizontal and vertical lines that define columns and rows. Elements can be placed onto the grid within these columns and rows to create the page layout.

CSS Grid Layout introduces a two-dimensional grid system to CSS.

Browser support for CSS Grid is good across all modern browsers, with only Internet Explorer having partial support.

Creating the Grid container

We create a grid container using the display property on an element. As soon as we do this, the children of the element become grid items.

    

.container {
    display: grid;
}

Our HTML will look something like this:

    

<div class="container">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
</div>

All the direct children are now grid items. The grid has created a single column for the grid items.

Grid Tracks

We define rows and columns on our grid with the grid-template-columns and grid-template-rows properties. These define grid tracks, a space between any two lines on the grid. The grid tracks can be defined using any length unit.

CSS Grid introduced the fr unit to represent a fraction of the available space in a grid container.

    

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}

This can also be written using the repeat() notation, to repeat all or a section of the track listing.

    

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Here we are setting the columns so that the three columns have equal space.

Gutters

Gutters, sometimes referred to as alleys, between grid cells can be created using the column-gap and row-gap properties. The shorthand gap property can also be used.

When the CSS Grid was first shipped in browsers, these properties were prefixed with grid- but as browsers are being updated to remove this prefix, this will be maintained as an alias making them safe to use.

    

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    column-gap: 1em;
    row-gap: 1em;
}

This can also be written like so:

    

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1em;
}

We can see that the gap between each cell is 1em between each row and column.

Setting the height of rows

We may want to give our tracks a minimum height when building a grid allowing the grid to expand to fit any content that is added. In this example, we may want our rows to never be smaller than 30px in height, but to automatically increase in height if needed.

    

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: minmax(30px, auto);
}

We can use the minmax() function for this within the grid-auto-rows property. This means the rows will automatically have a minimum height of 30px and a maximum set to auto. Using auto means that the height of the row will stretch to give space for the tallest item in a cell.
