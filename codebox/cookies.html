Cookies

Cookies are small text files stored on a computer used to store information in web pages. When a browser requests a web page from a server, cookies belonging to the page are added to the request.

The cookie property of the Document object lets you read and write cookies.

Write a new Cookie

We can create a cookie using document.cookie and set the name and value of this cookie within a string. Cookies are saved in name/value pairs.

    

document.cookie = "username=Dan Jackson";

We can also add an expiry date. By default, a cookie is deleted when the browser is closed.

    

document.cookie = "username=Dan Jackson; expires=Thu, 31 Dec 2020 12:00:00 GMT";

Return all cookies

We can use the cookie property to return all cookies in a single string.

    

let cookies = document.cookie;
