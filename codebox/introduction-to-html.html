Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

This is an HTML tag:

    

<p>

This is an HTML element with an opening and closing tag. Content is often placed between the two tags:

    

<p>HTML element</p>

This is an HTML self-closing tag as it does not require a closing tag to be valid:

    

<img src="image.jpg" width="200" height="100" alt="Example image">

HTML tags such as <img> can be self-closing.

<!DOCTYPE> declaration

The HTML declaration must be the very first thing in your HTML document. This is not an HTML tag but is an instruction to the browser about the version of HTML being used.

    

<!DOCTYPE html>

HTML lang attribute

Declaring the language attribute on the HTML tag enables a screen reader to read out text with the correct pronunciation.

    

<html lang="en">
<html lang="de">
<html lang="fr">

<html lang="en-GB">
<html lang="en-US">

<title>

The <title> tag defines the title of the HTML document. This is a requirement of all HTML documents. This tag is placed inside the <head> section of your HTML.

    

<title>Example page title</title>

Paragraphs

The <p> tag defines a paragraph. With a closing </p> tag, this forms a paragraph element.

    

<p>This is a paragraph.</p>
<p>This is yet another paragraph.</p>

This is a paragraph.

This is yet another paragraph.

<abbr>

The <abbr> tag defines an abbreviation using the title attribute.

    

<p>The <abbr title="World Health Organisation">WHO</abbr> was founded in 1948.</p>

The WHO was founded in 1948.

<detail> & <summary>

The <detail> element will hide its content by default. When the user clicks the heading, it expands to reveal the content inside. It can also be given a custom heading using the <summary> elment. It's supported by all major browsers apart from Internet Explorer (IE).

Click me

Any content inside is hidden by default.

    

<details>
    <summary>Click me</summary>
    <p>Any content inside is hidden by default.</p>
</details>

<dfn>

By wrapping a word or phrase in the <dfn> element, it indicates that this term is being defined. When placed inside a <p>, <dl>, or <section> element, the content surrounding it is treated as a definition. An id attribute can be used to link back to it whenever it next appears in the page.

Semantic markup is HTML that provides meaning to content.

    

<p><dfn>Semantic markup</dfn> is HTML that provides meaning to content.</p>

<meter>

For numerical data that has a defined range, the <meter> element can provide a visual indication of what the data means in context. The browser can conditionally style the element for high or low values.

    

<meter value="6.4" min="0" max="10" low="4" high="9" optimum="7.5"></meter>

<data>

The <data> element provides a hook for machines to understand meaning. The element may contain a value, such as as a product ID or ISBN. For date or time values, the <time> element should be used.

Practical Web Inclusion and Accessibility

    

<p><data value="9781484254516">Practical Web Inclusion and Accessibility</data></p>

<time>

Allows machines to parse time and dates within copy. A datetime attribute can be used that shows the date in a format that is easier for machines to parse.

Our event is happening on the 2nd April

    

<p>Our event is happening on the <time datetime="2021-04-02">2nd April</time></p>
