Building a Website: Getting started

With this guide we are going to build a website from the beginning to end.

Begin by installing a code editor to allow you to write your code. A few options are listed below:

  • Sublime Text
  • Atom
  • Visual Studio Code (VSCode)

Once installed, open your code editor and create a new HTML (HyperText Markup Language) file. All HTML files will have the file type of .html and we will save this in a folder somewhere on your machine where our website code will be located. In this example, our project folder will be called my-website.

Save the empty HTML file created as index.html inside the my-website folder.

The homepage of a website is often named index.html as most webservers recognise this file as the one to serve by default as the homepage.

Your file structure within your folder should look something like this:

    

my-website /
    index.html

Basic HTML structure

Within the index.html file, add the following HTML (HyperText Markup Language) code to build the basic template of our page. Once the code has been added save the file.

The file will now be a web page and can be opened in a web browser.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

</body>
</html>

When you open this page in a web browser no content will show, the page will be empty. Within the browser tab you will see the page title showing of My Webpage.

Any content we want to see on the page should be placed between the opening <body> and closing </body> HTML tags.

We will add the following heading to our HTML code in the index.html file.

    

<h1>Hello World</h1>

Once updated your code should look like the example below. Save the HTML file and refresh your web browser to see the heading appear.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

</body>
</html>

Next, under the heading add a paragraph as copy to the page. We will be using an opening <p> and closing </p> tag.

    

<p>This is an example paragraph.</p>

Our HTML code should now look like this:

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>

</body>
</html>

We can add multiple paragraphs if required.

    

<!DOCTYPE html>
<html>
<head>

    <title>My Webpage</title>

</head>
<body>

    <h1>Hello World</h1>

    <p>This is an example paragraph.</p>
    <p>This is yet another example paragraph.</p>

</body>
</html>
