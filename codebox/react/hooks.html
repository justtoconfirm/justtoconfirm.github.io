Hooks

Hooks allow function components to have access to state and other React features.

Introduction to Hooks

Hooks were added to React in version 16.8. Although Hooks generally replace class components, there are no plans to remove classes from React at this time. Hooks will not work in React class components.

Hooks are reusable functions and start with "use", for example "useState".

Hooks allow us to "hook" into React features such as state and lifecycle methods.

Import a Hook

Within our component, we first import the Hook we want to use. This example is using the useState hook.

	

import React, { useState } from 'react';

Set the Initial State

Within the function itself, using the useState Hook, we then set the default value of "red" in the state.

	

import React, { useState } from 'react';

const FavouriteColor = () => {
	
	const [color, setColor] = useState("red");

}

To improve the performance of our app, we can update our hook to only run once instead of each time the component renders/re-renders.

	

import React, { useState } from 'react';

const FavouriteColor = () => {

	const [color, setColor] = useState(() => {
		return "red";
	});

}
  • The first value, color, is our current state as a variable.
  • The second value, setColor, is the function that is used to update our state.

If we don't want to set an initial state, we can use an empty string:

	

import React, { useState } from 'react';

const FavouriteColor = () => {
	
	const [color, setColor] = useState("");

}

Update the State

We can then update the state within the component.

	

import React, { useState } from 'react';

const FavouriteColor = () => {
	
	const [color, setColor] = useState("red");

	return (
		
		<>
			<h1>My favourite colour is: { color }</h1>

			<button type="button" onClick={() => setColor("blue")}>Blue</button>
			<button type="button" onClick={() => setColor("green")}>Green</button>
			<button type="button" onClick={() => setColor("red")}>Red>/button>
			<button type="button" onClick={() => setColor("yellow")}>Yellow</button>
		</>

	);
}

export default FavouriteColor;

Building a Counter

Here we are building a simple component to increase or decrease a count using the useState hook.

	

import React, { useState } from 'react';

const Counter = () => {

	const [ count, setCount ] = useState(() => {
		return 4;
	});

	const increaseCount = () => {
		setCount(prevCount => prevCount + 1)
	}

	const decreaseCount = () => {
		setCount(prevCount => prevCount -1)
	}
	
	return (
		
		<>
			<button onClick={decreaseCount}> Decrease </button>

			<span>{ count }</span>

			<button onClick={increaseCount}> Increase </button>
		</>

	);
}

export default Counter;

Multiple States

We could create multiple state Hooks to track individual values.

	

import React, { useState } from 'react';

const Car = () => {
	
	const [brand, setBrand] = useState("Ford");
	const [model, setModel] = useState("Mustang");
	const [year, setYear] = useState("1964");
	const [color, setColor] = useState("red");

	return (
		
		<>
			<p>It is a { color } { brand } { model } from { year }.</p>
		</>

	);
}

export default Car;

We can also create a single Hook that holds an object:

	

import React, { useState } from 'react';

const Car = () => {
	
	const [car, setCar] = useState({
		brand: "Ford",
		model: "Mustang",
		year: "1964",
		color: "red"
	});

	return (
		
		<>
			<p>It is a { car.color } { car.brand } { car.model } from { car.year }.</p>
		</>

	);
}

export default Car;

We can use the JavaScript spread operator to update state. We add a function to update the colour in the state using a button.

	

import React, { useState } from 'react';

const Car = () => {
	
	const [car, setCar] = useState({
		brand: "Ford",
		model: "Mustang",
		year: "1964",
		color: "red"
	});

	const updateColor = () => {
		setCar(previousState => {
			return { ...previousState, color: "blue" }
		});
	}

	return (
		
		<>
			<p>It is a { car.color } { car.brand } { car.model } from { car.year }.</p>
			<button type="button" onClick={ updateColor }>Blue</button>
		</>

	);
}

export default Car;

The useEffect Hook

The useEffect Hook allows you to perform side effects in your components. The useEffect Hook runs on every render. We should always include the second parameter which accepts an array.

	

useEffect(() => {
	// Runs on every render
});

useEffect(() => {
	// Runs only on the first render
}, []);
