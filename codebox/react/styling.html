Styling

Apply styling using CSS, SCSS, and packages/libraries to your React applications and components.

Adding CSS

We can begin by creating a simple CSS stylesheet.

	

.btn {
	display: inline-block;
	text-align: center;
	padding: 1rem 2rem;
	text-decoration: none;
}

Within our React component, we can then import our CSS stylesheet to apply the styling.

	

import React from 'react';
import './Button.css';

const Button = () => {
	return (

		<button role="button" className="btn">Button</button>

	);
}

export default Button;

Adding CSS classes to React components

We can use the className prop to apply a class attribute to a React component. We cannot use class as it's considered a keyword.

	

<button role="button" className="btn">Button</button>

It is common for CSS classes to depend on the component prop or state.

	

let className = 'btn';

if (this.props.isActive) {
	className += ' btn--active';
}

return (

	<button role="button" className={ className }>Button</button>

)

styled-components

styled-components is a library for React and React Native that allows you to use component-level styles in your application written with a mixture of JavaScript and CSS.

styled-components allows React developers to write plain CSS in React components without having to worry about clashing of class names.

We first need to install the styled-components library. This can be done using npm.

bash -- 85x24

	npm install styled-components --save

As an example, we can update the earlier Button component to use the styled-components library.

	

import React from 'react';
import styled from 'styled-components';

const Button = styled.button`
	display: inline-block;
	text-align: center;
	padding: 1rem 2rem;
	text-decoration: none;

	&:hover {
		opacity: 0.9;
	}	
`;

const Button = () => {
	return (

		<Button role="button">Button</Button>

	);
}

export default Button;

We can also use the style-components library to create a global stylesheet for our React application.

Begin by creating a GlobalStyles.js file in the src directory of the app.

	

import { createGlobalStyle } from 'styled-components';

const GlobalStyles = createGlobalStyle`
	
	body {
		background-color: #fff;
	}

`;

export default GlobalStyles;

Open the src/App.tsx file and import GlobalStyles.

	

import React from 'react';
import GlobalStyles from './GlobalStyles';

const App = () => {
	return (

		<>
			<GlobalStyles />

			<h1>Hello World</h1>
		</>

	);
}

export default App;
