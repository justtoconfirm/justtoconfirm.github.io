Python

Python is an interpreted, high-level programming language used to build websites, automate tasks, and conduct data analysis.

Install Python

MacOS comes pre-installed with Python. To confirm the default version of Python installed, open your terminal and enter the command:

bash -- 85x24

	python --version

This is likely to be Python version 2.7, which is now deprecated by the Python developer community. It is recommended to install Python 3.x or Python 4.x as the latest versions.

It is best to install Python on Mac using Homebrew. Homebrew can be installed from within your terminal

bash -- 85x24

	/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

Once installed, run this command to check there's nothing missing to complete the setup:

bash -- 85x24

	brew doctor

We can install pyenv as a Python version manager to allow us to easily switch between multiple versions of Python.

bash -- 85x24

	brew install pyenv

To install a specific version of Python (version 3.10.0 in this example). We can use the global keyword to then set the version of Python once it's been installed.

bash -- 85x24

	pyenv install -v 3.10.0
	pyenv global 3.10.0
