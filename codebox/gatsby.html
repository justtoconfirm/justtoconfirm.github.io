Gatsby

Gatsby is an open-source framework built using React for creating websites and applications.

Requirements

You will need to have the following installed on your machine to install and run Gatsby:

  • Node 12.13.0 or later

You can use nvm (Node Version Manager) or nvm-windows to switch Node versions between projects.

Technoligies

Gatsby uses a few technologies under the hood:

  • React - a JavaScript library for building user interfaces and the UI components. This is used by Gatsby to build pages and structure content. View the React guide →
  • GraphQL - a query language that allows you to pull data into your website. It's the interface Gatsby uses for managing site data.

Install Gatsby

We can use Create React App to install React and to setup the development environment. This doesn't handle backend logic or databases, it just creates the frontend build pipeline, so we can use it with any backend we want.

This uses Babel and Webpack.

bash -- 85x24

	npm install gatsby-cli -g

Once installed, the file structure of the application should look like this:

	

my-app 
	|_node_modules /
	|_public /
		|_favicon.ico
		|_index.html
		|_logo192.png
		|_logo512.png
		|_manifest.json
		|_robots.txt
	|_src /
		|_App.css
		|_App.js
		|_App.test.js
		|_index.css
		|_index.js
		|_logo.svg
		|_reportWebVitals.js
		|_setupTests.js
	|_.gitignore
	|_package-lock.json
	|_package.json
	|_README.md

We can update the src/App.js file and the development server will reload the page.
