JavaScript Date & Time

JavaScript comes with a built-in Date object and related methods. The Date object is a built-in object in JavaScript that stores date and time. It provides a number of methods for formatting and managing the date and time.

To use the Date object you first need to create an instance of it. Without any arguments, the Date object corresponds to the current date and time (based on the system's settings).

    

// Create a new instance of the Date object and set a variable
const current = new Date();

Based on the current time and date, the output will be:

    

Tue Nov 17 2020 23:19:26 GMT+0000 (Greenwich Mean Time)

The date and time are derived from Unix time, also called Epoch time, which is a value consisting of the number of miliseconds that have passed since midnight on January 01, 1970.

We can use this same instance of the Date object to get a timestamp using the getTime() method:

    

current.getTime();      // 1605655345867

The value of 1605655345867 in this example is the number of milliseconds that have passed since Epoch time.
