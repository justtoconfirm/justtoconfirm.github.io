JavaScript Objects

Objects group together a set of variables and functions to create a model. In an object, variables and functions take on new names. In an object, variables become known as properties and functions are known as methods. Properties tell us about the object, methods represent tasks that are associated with the object.

Properties and methods have a name and a value. Within an object, the name is a key. Each key and value is separated by a comma and an object cannot have two keys with the same name. The value of a property can be a string, number, Boolean, array, or another object. The value of a method is always a function.

Creating Objects: Literal Notation

Literal notation is the easiest and most popular way to create objects in JavaScript. To create an empty object literal, use:

    

var obj = {};

The object is stored in a variable. In the example above, the object is stored in a variable called obj, so we would refer to it as the obj object.

This object represents a hotel. It has five properties and one method.

    

var hotel = {
    name: 'Quay',
    rooms: 40,
    booked: 25,
    gym: true,
    roomType: ['twin', 'double', 'suite'],

    checkAvailability: function() {
        return this.rooms - this.booked;
    }
};

The hotel object has a method named checkAvailability(). The this keyword is used to indicate that the method is using the rooms and booked properties of this object.

The function keyword can also be omitted when assigning methods to an object.

    

checkAvailability() {
    return this.rooms - this.booked;
}

Creating Objects: Constructor Notation

The new keyword and object constructor can be used to create a new empty object. Properties and methods can then be added to the object. The Object() constructor is a function that's part of the JavaScript language and is used to create objects. Once an object has been created, dot notation can be used to add the properties and methods.

To create an empty object using the Object() constructor function:

    

var obj = new Object();

Here we can use the Constructor Notation to create a hotel object and add properties and methods.

    

// Create a new blank hotel object
var hotel = new Object();

// Add properties to this object
hotel.name = 'Quay';
hotel.rooms = 40;
hotel.booked = 25;

// Add method to this object
hotel.checkAvailability = function() {
    return this.rooms - this.booked;
};

Sometimes you may need many objects to represent similar things. Object constructors can use a function as a template for creating objects. Here we are using a function to create a template with the object's methods and properties.

    

function Hotel(name, rooms, booked) {
    
    this.name = name;
    this.rooms = rooms;
    this.booked = booked;

    this.checkAvailability = function() {
        return this.rooms - this.booked;
    };

};

Here a function called Hotel will be used as a template for creating new objects that represent hotels. Like all functions, it contains statements. In this case, the statements add properties and methods to the object. This function has three parameters, each sets the value of a property in the object. The methods will be the same for each object created using this function template.

The name of the function usually begins with an uppercase letter to help remind developers to use the new keyword when they create an object using this template function.

The this keyword is used instead of the object name to indicate that the property or method belongs to the object that this function creates.

Instances of the Template

You can create instances of the object using a constructor function. The new keyword followed by a call to the function, creates a new object using the template. The properties for each object you create are given as arguments to be passed into the function parameters.

Here two objects are created to represent two hotels. Each object needs a different name. When the new keyword calls the constructor function, it creates a new object. Each time it's called, the arguments passed into the function are different because they are properties of two different hotels. Both objects automatically get the same method defined within the function - the checkAvailability() method.

    

var quayHotel = new Hotel('Quay', 40, 25);
var parkHotel = new Hotel('Park', 120, 77);

Even when many objects are created using the same constructor function, the methods stay the same because they access, update, or perform a calculation based on the data stored in the properties.

    

quayHotel.checkAvailability();  
parkHotel.checkAvailability();

This technique maybe used if your script contains a complex object that needs to be available but might not be used. The object is defined in the function, but only created if it's needed.

Accessing an Object

You can access the properties or methods of an object using dot notation. Square brackets can also be used. To access a property or method of an object, use the name of the object followed by a period - the member operator - then the name of the property or method you want to access.

Here we declare two variables to hold the name of the hotel and the number of vacant/available rooms.

    

var hotelName = hotel.name;
var roomsFree = hotel.checkAvailability();

Properties can also be accessed using square brackets. Methods cannot be accessed using this. This time the object name is followed by square brackets and the property name is inside.

    

var hotelName = hotel['name'];

Updating an Object

To update the value of a property inside an object, use dot notation once again but assing a new value using the assignment operator (=). Here the name of the hotel is updated to Park.

If the property you are attempting to update does not exist, it will be added as a new property to the object.

    

hotel.name = 'Park';

To delete a property from an object, use the delete keyword followed by the object name and name of the property.

    

delete hotel.name;

If you want to clear a value of a property, set the value to be a blank string.

    

hotel.name = '';
