An Introduction to PHP

PHP is a popular general-purpose server scripting language that is especially suited to web development.

Introduction to PHP

PHP (PHP Hypertext Preprocessor) scripts usually have the file extension of .php

The file extension of .php tells the web server to send the file to the PHP interpreter so that it can follow the instructions written in PHP code.

Many PHP pages mix HTML and PHP code. The PHP code is written between PHP tags. The opening and closing PHP tags, plus any PHP code they contain, are known as a PHP block.

PHP is a type of programming language known as a scripting language. Scripting languages are designed to run in a particular environment. PHP was created to work with a PHP interpreter on a web server. An individual PHP page is often called a script.

  • Opening tag indicates that the PHP interpreter must start processing the code before sending any content back to the browser.
  • The closing tag indicates that the PHP interpreter can stop processing the code.
	

<?php	?>

Code Blocks

PHP uses curly braces to surround related sets of statements in our PHP code. The braces and the statements inside them form a code block.

	

{
	// Curly braces indicate the
	// start and the end of a
	// code block.
}

Braces let the PHP interpreter known where a code block starts and ends. The block allows the PHP interpreter to run, skip, or repeat the statements that they contain. There should be no semi-colon at the end of the code block.

Variables

Variables store data that can change each time a PHP page is requested.

  • The name describes the type of data that the variable holds.
  • The value is the value the variable should hold when the page is requested.

To create a variable and store a value in it, you need:

  • A variable name which must start with a dollar sign, followed by one or more words that describe the type of information the variable can hold.
  • An equals sign which is known as the assignment operator because it assigns a value to the variable name.
  • The value you want the variable to hold.
  • Underscores are allowed in the name of a variable, no dashes or periods however.
	

$name = 'Dan';
$age = 26;

Once a variable has been declared and a value assigned to it, the variable name can be used in the PHP code. When the PHP comes across a variable name, it replaces the name with the value it holds.

	

echo $name;

Array

To create an associative array, give each element (item) in the array a key that describes the data it holds.

	

$users = [
		'name' => 'Dan',
		'age' => '26',
		'country' => 'United Kingdom',
];

In this example, the key would be something like name, the operator would be => and the value would be Dan.

To access an element from an associative array:

	

$users['name'];

When an array is created, if a key is not assigned for each element, the PHP interpreter will assign it a number called an index number. These index numbers start at zero (0), not one (1).

	

$shopping_list = ['bread', 'milk', 'cheese',];

This array above can also be created using the syntax below:

	

$shopping_list = array(
	'bread', 
	'milk', 
	'cheese'
);

To access the third item in the array:

	

$shopping_list[2];

Echo

Instead of writing the echo statement, we can use the shorthand.

	

// Longhand
<?php echo $name; ?>

// Shorthand
<?= $name; ?>

Conditional Statements

A condition always results in, or evaluates into, a boolean value of either true or false. The result determines which code block is run.

	

if ($logged_in === true) {
	$link = '<a href="member.php">My Profile</a>';
} else {
	$link = '<a href="login.php">Login</a>';
}

We can also use a ternary operator to check a condition then provide one value to use if the condition is true and another value if the condition is false.

	

<?php

		$stock = 5;

		$message = ($stock > 0) ? 'In stock' : 'No stock';

?>
