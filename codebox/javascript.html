JavaScript

JavaScript (JS) is a high-level programming language that conforms to the ECMAScript specification.

Adding JavaScript to HTML

Once a HTML document has been created, add a <script> tag before the closing </body> tag to add a JavaScript script to your HTML.

	

	<script src="js/script.js"></script>

</body>
</html>

The script maybe minified to reduce the file size and better improve performance. Minification will reduce the number of characters in your file to reduce the number of bytes of code within your JavaScript. A minified file may have the following naming convention:

	

	<script src="js/script.min.js"></script>

</body>
</html>

Functions

Functions are declared (created) using the function keyword.

	

function functionName() {
	
	// Code to be executed here

}

We can also write functions using the ES6 arrow syntax.

	

const functionName = () => {
	
	// Code to be executed here

}

Nothing will happen and no code will execute until we call (invoke) the function.

	

const functionName = () => {
	
	// Code to be executed here

}

// Call the function
functionName();

Function Parameters

Parameters allow our functions to be flexible. We can use parameters to pass values into functions. Within the following function, we use two parameters to calculate the area.

	

const area = (width, height) => {
	
	return width * height;

}

We pass values into the function parameters using arguments. We assign values when we invoke (call) our function.

	

const area = (width, height) => {
	
	return width * height;

}

area(5, 2);		// 5 * 2

Default parameters were introduced as part of the ES6 syntax. This allows us to set a default value if an argument is not provided when the function is called.

	

const area = (width = 5, height = 4) => {
	
	return width * height;

}

area();		// 5 * 4

Objects

A Literal notation is the easiest way to create an object in JavaScript. The object is stored in a variable. In this example, we would refer to this as the obj object.

	

const obj = {};

Properties can be added to objects. A property is similar to a variable and is a key/value pair.

	

const obj = {
	key: value
};

Methods can also be added to objects. A method is similar to function within an object.

	

const obj = {
	key: value,

	method() {
		// code here
	}	
};

We can use dot notation to access the value of an object property.

	

obj.key;

A method can also be invoked (called) by referencing the object's name with the name of the method.

	

obj.method();

Using the object literal notation, we can build an object that represents a hotel.

	

const hotel = {
	name: 'The Grand Hotel',
	rooms: 250,
	roomsBooked: 189,
	gym: true,
	roomType: ['single', 'double', 'suite', 'penthouse'],

	checkRoomsAvailable() {
		return this.rooms - this.roomsBooked;
	}	
};

We can check the number of rooms available in our hotel using the checkRoomsAvailable() method.

	

hotel.checkRoomsAvailable();

Destructuring

Object destructuring allows you to create a variable for each property allowing you to access properties of objects more easily and reducing the amount of code we need to write. This is a feature of the ES6 JavaScript syntax.

Here we have a simple object that represents a task.

	

const task = {
	id: 1,
	title: 'Learn JavaScript',
	date: '01/01/2022'
};

We can now destructure the object using a single line of code to create new variables from each property in our object.

	

const { id, title, date } = task;

We can then use these variables in our code. Here's the full code example.

	

const task = {
	id: 1,
	title: 'Learn JavaScript',
	date: '01/01/2022'
};

const { id, title, date } = task;

console.log(title);		// Learn JavaScript

If the object has nested objects, we just need to update our destructure code slightly.

	

const task = {
	id: 1,
	title: 'Learn JavaScript',
	date: '01/01/2022',
	author,
	author: {
		firstName: 'Dan',
		lastName: 'Smith'
	}
};

const { id, title, date, author: { firstName, lastName } } = task;

console.log(title);		// Learn JavaScript

We can use a template literal to log the firstName and lastName property values.

	

console.log(`${firstName} ${lastName}`);

We can also use destructuring to access the properties of an object. For example we can get the length of a String using destructuring.

	

const {length} = 'Learning JavaScript is fun';

console.log(length);		// 26

Arrays

An Array is an object in JavaScript that is used to store data. They are useful for storing multiple values in a single variable.

An empty array can be declared like this:

	

const array = [];

The length property of array can be used to output the total number of items in our array.

	

const skills = ['HTML', 'CSS', 'JavaScript', 'PHP'];

skills.length;		// 4

A for loop also uses the length property to loop over the array.

	

const skills = ['HTML', 'CSS', 'JavaScript', 'PHP'];

let i;		// Declare a variable so not to declare on each loop

for (i = 0; i < skills.length; i++) {
	console.log(skills[i]);
}

The ES6 for loop syntax:

	

const skills = ['HTML', 'CSS', 'JavaScript', 'PHP'];

let i;		// Declare a variable so not to declare on each loop

for (i of skills) {
	console.log(i);
}

We can use the forEach() method to loop over each item contained within an array.

	

const skills = ['HTML', 'CSS', 'JavaScript', 'PHP'];

skills.forEach(skill => {
	console.log(skill);
});
