What is code-splitting

Code splitting is a great feature of Webpack that allows you to split your code into various 'bundles' which can then either be loaded on demand, or all at the same time. With this you can then control the load prioritasion too which can have a big impact on improving the load time.

Two different types of splitting

  • Bundle splitting: creating more, smaller files (but loading them all on each network request anyway) for better caching
  • Code splitting: dynamically loading code, so that users only download the code they need for the part of the site that theyâ€™re viewing.
