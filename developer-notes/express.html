Express

Express, often referred to as ExpressJS, is a framework for Node (NodeJS). To install Express, you will need Node (and NPM - Node Package Manager) installed, open the terminal and run the following command:

        

$ npm install express --save

To install Express temporarily and not add it to the dependencies list within your package.json file:

        

$ npm install express --no-save

This will install Express as a dependency within your project. You should already have a package.json file within your project dirctory. If not, create one:

        

# Create package.json
$ npm init -y
