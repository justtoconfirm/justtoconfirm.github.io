Python Basics

Python is a general-purpose, interpreted, object-oriented, high-level programming language created by Guido van Rossum. Python is processed at runtime by the interpreter, you do not need to compile your program before executing it. This is similar to PHP.

Python is an object oriented programming language. Almost everything in Python is an object, with properties and methods.

To check the version of Python installed, open the terminal and enter the command below.

        

$ python --version

Let us write a simple Python program in a script. Python files have the extension '.py'. Type the following code in a file and name it as 'test.py'.

        

print "Hello, World"

Now, in the terminal, run the program.

        

$ python test.py

Install & Setup

It is recommended that you install the latest Python 3.x version (Python 3). You may also need pip installed.

        

$ pip --version

Pipenv is a dependency manager for Python projects, similar to how NPM (Node Package Manager) works with NodeJS. While pip can install Python packages, Pipenv is recommended as its a higher-level tool that simplifies dependency management.

We use pip to install Pipenv.

        

$ pip install --user pipenv

Variables

Python supports different types of variables (datatypes). You do not need to specify the datatype of a variable, you can simply assign any value to a variable. Python will determine the datatype based on the value you assign to the variable.

        

x = 3               # A whole number
y = 3.14            # A floating number
name = "Python"     # A string

print(x)
print(y)
print(name)

combine = name + " " + name
print(combine)

By convention variables are often camel cased, meaning the first letter is small and the next words are uppercase.

        

daysInYear = 365
daysInMonth = 30
numberOfFiles = 5

Functions

To group sets of code you can use functions. Functions are small parts of repeatable code. They can be seen as executable code blocks and can be used more than once.

To declare a function in Python, the def keyword is used.

        

def currentYear():
    print('2018')

// Call the function
currentYear()

In this example below, we have a function and parameters x and y. In this example we also use the print() function. The output is passed to the print() function using the return keyword.

        

def multiplyTwoNumbers(x, y):
    return x * y

// Call the function
print(multiplyTwoNumber(3, 4))

Functions can return variables. Sometimes a function makes a calculation of has some output. This can be given to the program with a return statement.

Lists

List can be seen as a collection that can hold many variables. They resemble physical lists and contain a number of items. They are similar to Arrays in other programming languages.

Here we are creating an empty list. To define an empty list you should use brackets. Brackets is what tells Python that the object is a list.

        

list = []

To access an item in a list, add the id (index) between the brackets, such as list[0], list[1], and so on. Python starts counting at zero, that means the first element in the list has an index of zero.

        

// Get the first item from the list
print(rating[0])

If we wanted to get the third item in the list, we would use the index of 2.

        

// Get the third item from the list
print(rating[2])

Here the list contains numerical values:

        

ratings = [ 3, 4, 6, 3, 4, 6, 5 ]

// Output the list
print(ratings)

You can interact with the list using a for loop.
