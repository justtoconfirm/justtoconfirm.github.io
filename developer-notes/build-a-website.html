Build a Website

In this guide we are going to go through the step-by-step process of building a website.

We are going to start with the initial HTML (Hypertext Markup Language) structure. Open your code editor and add the following code:

        

<html>
<head>

</head>
<body>

</body>
</html>

Save this file as index.html Your file needs to have the filetype of .html for it to be saved as a HTML webpage. Open the page within your web browser and you will see that it's empty and has no content.

Remember to save your code each time you make changes.

Adding a page title

We are going to provide our page with a title of 'My Website'. We will use the <title> HTML tag to apply a title to our page. We will need an opening <title> and a closing </title> tag. Together they create a title element. Between the opening and closing tags, add the page title text.

Add the <title> element between the opening <head> and closing </head> tags.

        

<html>
<head>
    <title>My Website</title>
</head>
<body>

</body>
</html>

Adding content

Our webpage currently has no content. Any content you want to display on the page should be placed between the opening <body> and closing </body> tags. Let's add some simple text of 'Hello World' to our page.

        

<html>
<head>
    <title>My Website</title>
</head>
<body>
    <p>Hello World</p>
</body>
</html>

You may notice that our example uses <p> and </p> tags. They are used to create paragraphs.

<!DOCTYPE>

The <!DOCTYPE> declaration must be the very first thing in your HTML document, before the <html> tag. This is not an HTML tag, but an instruction to the web browser about the version of HTML being used.

We are going to use the HTML5 Doctype for our HTML document. This needs to be placed on the first line of your HTML code, just before the <html> tag.

        

<!DOCTYPE html>

Semantic Structure

We are going to use HTML5 tags to ensure our page structure is semantic. Let's begin by adding a <main> tag to our HTML code. This needs to be placed within the <body> section of our code and wrap around any content within the page.

Spacing has been added but only to demonstrate the changes.

        

<!DOCTYPE html>
<html>
<head>

    <title>My Website</title>

</head>
<body>
<main>
    
    <p>Hello World</p>

</main>
</body>
</html>

We can add an attribute to the <main> element to ensure it is supported in browsers such as Internet Explorer 11 (IE11) and below. HTML attributes are used to define a property of an element. They consist of pairs of properties and values. Attributes should be placed on the opening HTML tag.

We will add an ARIA (Accessible Ricj Internet Applications) attribute of "role" onto the <main> tag to ensure it is accessible. Screen readers will then be able to understand the semantic meaning of the <main> element once the attribute has been added.

        

<main role="main">

Setting the Language

Within the <html> tag, we can use the "lang" attribute to set the language for our document. We will be setting the language to English, so we need to use the value "en".

        

<html lang="en">

This is our HTML code so far:

        

<!DOCTYPE html>
<html lang="en">
<head>

    <title>My Website</title>

</head>
<body>
<main role="main">
    
    <p>Hello World</p>

</main>
</body>
</html>

Meta Tags

Next, we need to add at least one meta tag to our HTML code to set the character encoding for the page. Place the following meta tag just after the opening <head> tag and before the <title> element.

        

<meta charset="utf-8 />

Our HTML code should now look like this:

        

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <title>My Website</title>

</head>
<body>
<main role="main">
    
    <p>Hello World</p>

</main>
</body>
</html>

Adding Links

We want our HTML page to link to other pages within our website or to external websites. Links (sometimes called Hyperlinks) can be used to click to a new page or navigate through a website. In HTML, links are defined with the <a> tag.

        

<a href="url">Link text</a>

Let's add a link that opens Google. The link should be placed within the <body> section of your code. The "href" attribute is the URL of the website or page you wish to link to navigate to when clicked. The Text between the opening <a> and closing </a> tags is the text for the link that appears on the page.

        

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <title>My Website</title>

</head>
<body>
<main role="main">
    
    <p>Hello World</p>
    <a href="https://www.google.co.uk">Google</a>

</main>
</body>
</html>
