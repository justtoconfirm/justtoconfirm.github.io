Regex

Regex, or Regular expression, is a string of text that allows you to create patterns that help, match, locate, and manage text. If you need to work with text or need to parse large amounts of data, Regex is ideal.

In JavaScript, regular expressions are also objects. These patterns are used with the exec and test methods of RegExp, and with the match, replace, search, and split methods of String.

Regular Expression Literal

A Regular expression literal consists of a pattern enclosed between slashes. They provide compilation of the Regex when the script is loaded. This can improve performance if the expression remains constant.

        

var regex = /ab+c/;

We can also call the constructor function of the RegExp object, as follows:

        

var regex = new RegExp('ab+c');

Use this method if you know the pattern will be changing, or if you don't know the pattern and are getting it from another source, such as user input.

Writing a Regex pattern

A Regex pattern is composed of simple characters, such as /abc/, or a combination of simple and special characters, such as /ab*c/.

Simple patterns are constructed of characters for which you want to find a direct match. For example, pattern /abc/ matches characters in a combination within a string only when 'abc' occur together in that order. Such a match would succeed in the strings: "Hi, do you know your abc's?". There is no match with the string "Grab crab" because while it contains 'ab c', it does not contain the exact substring of 'abc'.

In the following example, the script uses the exec method to find a match in a string.

        

var myRegex = /d(b+)d/g;
var myArray = myRegex.exec('cdbbdbsbz');

Regex can be used to verify form input. It can also be used for "search-and-replace" functionality performed on text.
