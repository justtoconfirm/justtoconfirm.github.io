JavaScript Array

In JavaScript, the array data type consists of a list of elements. There are many useful built-in methods available for developers to work with arrays. Methods that modify the original array are known as mutator methods and methods that return a new value or representation are known as accessor methods. A third class of array methods, known as iteration methods, which are methods that operate on every item in an array, one at a time, also exist. These methods are closely associated with loops.

Array methods are properly written out as Array.prototype.method(), as Array.prototype refers to the Array object itself.

We will use the following Array for all of our examples:

        

var fish = [ "piranha", "barracuda", "cod", "eel" ];

Iteration Methods

Our examples may use the arrow function syntax.

forEach()

The forEach() method calls a function for each element in an array.

        

fish.forEach(individualFish => {
   console.log(individualFish); 
});

Once we call this function, we'll receive the following output:

        

piranha
barracuda
cod
eel

Another way to do this is using the for loop and testing it against the length property of the array.

        

// Loop through the length of the array
for (var i = 0; i < fish.length; i++) {
    console.log(fish[i]);
}

filter()

The filter() method creates a new array with the elements that pass the result of a given test. We could use filter() to return a new array containing only the items in a list that start with a specific letter. We can utilize string indexing to call the first item (or letter) in each string item of the array.

We are testing which items in the array have a "p" at the 0 index (the first letter), and assigned the result into a new variable. filter() is an iteration method and does not mutate the original array.

        

// Filter all fish that start with "p" into a new list
var filteredList = fish.filter(creature => {
   return creature[0] === "p"; 
});

filteredList;       // ["piranha"]

find()

The find() method returns the first value in an array that passed a given test.

As an example, we will create an array of cars. We will then use the find() method to test if any of the cars in the array are from Germany.

        

var cars = [ "Volvo", "Vauxhall", "BMW", "Fiat", "Audi", "Mercedes-Benz" ];

// Check if a given value is a German car
const isGerman = german => {
    return [ "Mercedes-Benz", "Audi", "BMW" ].includes(german);
}

cars.find(isGerman);        // "BMW"

Since "BMW" was the first entry in the array to satisfy the test in the isGerman() function, it is the first value to be returned.

findIndex()

The findIndex() method returns the first index in an array that passes given test.

We can use the same code as before, but find the index number instead of the value of the first match. If the test is not satisfied, findIndex() will return -1.

        

cars.findIndex(isGerman);       // 2
