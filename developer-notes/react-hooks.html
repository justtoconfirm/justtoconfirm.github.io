React Hooks

Hooks are a new feature within React that lets you use state, lifecycle, and other React features, without the need to write a class. This allows us to add features from React classes to functional components. This doesn't mean we have to stop using classes, it just gives us more ways to code using React.

On February 6th 2019, React 16.8 was released and React Hooks are now available in a stable release.

Current Components

Lets demonstrate how components are created currently in React. To begin, a simple functional component:

        

const Example = () => {
    return <div>I'm a simple component</div>
}

And now a class component:

        

class Example extends Component {
    render() {
        return <div>I'm a class component</div>
    }
}

useState()

A new function within React, useState(), is the first "Hook". Here is an example:

        

import React, { useState } from 'react';

function Example() {
    // Declare a new state variable, which we'll call "count"
    const [count, setCount] = useState(0);

    return (
        <div>
            <p>You clicked {count} times</p>
            <button onClick={() => setCount(count + 1)}>Click me</button>
        </div>
    );
}

This example renders a counter. When you click the button, it increments the value.
