Bootstrap

Bootstrap is one of the most popular, open-source, front-end frameworks on the web. It was officially released in 2011 and is now one of the most stable and responsive frameworks available.

The Bootstrap official site and documentation.

Bootstrap provides a solid foundation for any website irrespective of project size. It contains Reboot, which is based on Normalize.css, and helps level out browser differences for various page elements.

Some projects such as Bootstrap, which for the time being has a dependency on jQuery, is something that needs to be considered if you intend to use Bootstrap for new projects.

The bootstrap.bundle.js and bootstrap.bundle.min.js file(s) include popper.js but not jQuery.

Setup

Using BootstrapCDN, provided by StackPath, is a quick and simple way to include Bootstrap into your project. Begin by copying and pasting the following <link> into your <head> before all other stylesheets to load the Bootstrap CSS.

        

<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.2.1/css/bootstrap.min.css" integrity="sha384-GJzZqFGwb1QTTN6wy59ffF1BuGJpLSa9DkKMp0DgiMDm4iYMj70gZWKYbI706tWS" crossorigin="anonymous">

Alternatively, you can simply download Bootstrap from the downloads page. At the time of writing, the latest version of Bootstrap is version 4 (4.2.1).

Many of the components within Bootstrap requires the use of JavaScript to function. Specifically, they require jQuery, Popper.js, and the custom Bootstrap JavaScript plugins.

Browser Support

Bootstrap supports the latest, stable releases of all major browsers and platforms including Internet Explorer 10/11 and Edge. Autoprefixer is included to handle intended browser support via CSS prefixes which uses Browserslist to manage these browser versions.

IE9 (Internet Explorer 9) and down is not supported and some CSS3 properties and HTML5 elements are not fully supported in IE10, or may require prefixed properties for full functionality.

If you requires support for IE8 or IE9, use Bootstrap 3.

Grid

The Grid within Bootstrap is built with Flexbox and is easy to use, logically dividing the screen into 12 columns. Developers now have the native CSS Grid Layout should they choose to use it for layout building, but the Bootstrap Grid component can still be handy for quick prototyping.
