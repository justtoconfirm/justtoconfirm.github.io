Parcel

Parcel is a web application bundler with zero configuration needed. Many web application, or module, bundlers are available including Webpack and Browserify, but they are built around configuration and plugins. Parcel is designed to have zero configuration, just point it at the entry point of your application and it does the right thing.

To install Parcel, Yarn or NPM (Node Package Manager) can be used:

        

$ yarn global add parcel-bundler 

$ npm install parcel-bundler -g

Parcel can take any type of file as an entry point, but a HTML or JavaScript file is a good place to start. Parcel also has a development server built in which will automatically rebuild your app as you change files and supports hot module replacement for fast development. Just point it at your entry file:

        

$ parcel index.html

Now open http://localhost:1234/ in your browser.
