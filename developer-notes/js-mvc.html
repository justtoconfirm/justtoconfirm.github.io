JavaScript MVC

Writing a simple application in plain JavaScript using the model-view-controller (MVC) architectural pattern. This will be a todo app allowing you to CRUD (Create, Read, Update and Delete) todos within the browser. It consists of an index.html, style.css and script.js files, so is simple and free from any framework/library dependencies.

MVC is one possible pattern for organising your code, and it's fairly popular.

  • Model - Manages the data of an application
  • View - A visual representation of the model
  • Controller - Links the user and the system

The model is the data. In the todo application, this will be the actual todos and the methods that will add, edit, or delete them.

The view is how this data is displayed. In this application, this will be rendered HTML in the DOM and CSS.

The controller connects the model and the view. It takes user input, such as clicking or typing, and handles callbacks for user interaction.

The model never touches the view, the view never touches the model. The controller connects them.

Setup

This is going to be a fully JavaScript app, which means everything will be handled through JavaScript and the HTML will only consist of a single root element in the body.

        

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">

    <title>Todo App</title>
</head>
<body>

<div id="root"></div>

<script src="dist/script.js"></script>

</body>
</html>

You can add your style.css stylesheet to add style to the application. For now, this will be omitted to focus on the JavaScript. You will need to setup Babel to allow JavaScript classes to be writtten.

Building the App

We're going to create a really nicd and simple to understand what class pertains to what part of MVC. I'll make a Model, View and Controller class, which takes in the model and view. The app will be an instance of the controller.

        

class Model {
    constructor() {}
}

class View {
    constructor() {}
}

class Controller {
    constructor(model, view) {
        this.model = model;
        this.view = view;
    }
}

// App is an instance of the controller
const app = new Controller(new Model(), new View())
