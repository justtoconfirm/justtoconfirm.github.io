JavaScript: Objects

An object in JavaScript is a data type composed of a collection of names/keys and values. These names/keys and value pairs can consist of properties that may contain any data type - including strings, numbers, and Booleans - as well as methods.

Objects are an extremely useful and versatile feature of the JavaScript programming language and are some of the main building blocks of writing code in JavaScript. Objects are a practical way to organise related data and functionality within your code. To-do lists, shopping carts, user accounts and locations on a webpak are a few of the many examples of real-world JavaScript objects you may encounter.

Methods are functions contained within an object.

Objects in JavaScript are standalone entities that can be likened to objects in real life. For example, a book may be an object which you would describe by the title, author, number or pages, and genre. A car might be an object you would describe by the colour, make, model and horsepower.

JavaScript arrays are also a type of object.

Objects are an integral and foundational aspect of most JavaScript programs. A common use case is a shopping cart that could consist of an array of many objects containing information for each item, such as name, price, and quantity. A to-do list is another example as is a user account object that may contain such data as usernames, passwords, and e-mail addresses.

An object is a JavaScript data type, just as a number or string is also a data type. As a data type, an object can be contained within a variable.

There are two ways to construct an object in JavaScript:

  • The object literal, which uses curly brackets: {}
  • The object constructor, which uses the new keyword

First, the object literal.

        

// Initialize object literal with curly brackets
const objectLiteral = {}

In this next example, we'll use the object constructor.

        

// Initialize object constructor with the new Object
const objectConstructor = new Object()

Both of these approaches will create an empty object.

You may notice the this keyword being used. When using this inside of an object, it refers to the current object.

Using object literals is the more common and preferred method for creating objects as it has less potential for inconsistencies and unexpected results.

A simple object

We can create an example object contained in the variable of person, to describe someone.

        

// Initialize the person object
const person = {
    name: 'Dan',
    gender: 'Male',
    age: 34,

    greet: function() {
        return `Hi, my name is ${this.name}`
    }
}

Our new object is person which has three properties. Each property consists of a name:value pair, also known as a key:value pair. One of the property names is gender which is linked to the property value "Male", a string. The object has one method, named greet.

Within the greet method, the this keyword is used. The this keyword is used inside of the object to refer to the current object, in this case, the person object.

Sending person to the console will print out the entire object.

        

console.log(person);        // {name: "Dan", gender: "Male", age: 34, greet: f}

Accessing object properties

There are two ways to access an object's properties:

  • Dot notation: (.)
  • Bracket notation: [ ]

If you want to retrieve the property value of name from our object, we can do so with object dot or bracket notation, like so:

        

// Dot notation
person.name

// Bracket notation
person['name']

In order to retrieve a method within an object, you would call it in much the same way you would call a regular function just attached to the object variable.

        

person.greet()

The example above will return the string value for the object method greet().

Looping through object properties

JavaScript has a built-in type of for loop that is specifically meant for iterating over the properties of an object. This is known as the for...in loop. This loop can be used to traverse through all the properties of our object and print them to the console.

Using bracket notation, we can retrieve the property value as a variable, in this case key.

        

// Iterate through properties of the person object
for (let key in person) {
    console.log(person[key])
}

The output when using the loop should be:

        

Dan
Male
34

We can also retrieve the property name itself using just the first variable in the for...in loop. We have used a string method to convert the key values to upper case.

        

// Get keys and values of the person properties
for (let key in person) {
    console.log(key.toUpperCase() + ':', person[key])
}

The output when using the loop should be:

        

NAME: Dan
GENDER: Male
AGE: 34

Another useful enumeration method is the Object.keys() method which will return an array of the object's keys.

        

// Initialize method on the person object to return property keys
Object.keys(person)         // ["name", "gender", "age", "greet"]

The Objects.keys method allows us to work with the keys or names of an object as an array so you can leverage all of the methods available to JavaScript arrays.
