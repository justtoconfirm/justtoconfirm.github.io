NPM (Node Package Manager)

NPM stands for Node Package Manager. It is the largest software package library in the world with over 600,000 packages. NPM is installed with Node.js which includes a Command Line Interface (CLI) for NPM allowing developers to interact with packages locally on their machines.

The NPM repository can be located at: https://www.npmjs.com/

You may also want to check out Yarn as an alternative to NPM.

NPM came bundled with Node from version 0.6.3 and above.

Once Node.js has been installed, NPM should also be installed. To confirm the version of NPM installed open your terminal and enter the following command:

        

$ npm --version

If you want a special treat for Christmas, open your terminal and use the following command:

        

$ npm xmas

Install a package

The NPM install command installs a package and any packages that it depends on. If the package has a package-lock or shrinkwrap file, the installation of the dependencies will be driven by the versions listed in the package-lock or shrinkwrap files.

The npm-shrinkwrap.json file will take precedence over the package-lock.json file. The package-lock.json file will be generated when you install an NPM package.

The syntax used within the terminal to install an NPM package:

        

$ npm install <package-name-here>

# Shorthand syntax
$ npm i <package-name-here>

Install a global package

To install a global package, use the -g flag:

        

$ npm install <package-name-here> -g

# Shorthand syntax
$ npm i <package-name-here> -g

To list the NPM packages that are installed as global packages, this command can be used:

        

$ npm list -g --depth 0

Install a package as a devDependency

To install a package as a devDependency, use the --save-dev flag:

        

$ npm install <package-name-here> --save-dev

# Shorthand syntax
$ npm i <package-name-here> --save-dev

You will need to create a package.json file within your project folder/directory to install an NPM package as a devDependency.

Install a package as a Dependency

To install a package as a Dependency, use the --save flag:

        

$ npm install <package-name-here> --save

# Shorthand syntax
$ npm i <package-name-here> --save

You will need to create a package.json file within your project folder/directory to install an NPM package as a Dependency.

To view locally installed NPM packages, use the command:

        

$ npm ls

The package.json file

Begin by creating or opening a directory and generating a package.json file. This will allow you to install an NPM package as a dependency or devDependency.

        

# Create a directory
$ mkdir my-project
# Change directory into the my-project folder
$ cd my-project

# Generate a package.json file
$ npm init -y

Once you've created the package.json file in your project, you can begin to install packages using NPM locally. These will be listed within your package.json file once installed and a node_modules folder will be generated inside the root of your project folder.

It's worthwhile creating a .gitignore file in the root of your project folder when using NPM packages to prevent the node_modules directory from being added to your repository when using version control.

Update the package.json version

When the package.json file has been created the default version is 1.0.0 but this version can be updated/changed using NPM. When changes occur, we can update the version as a patch, minor or major patch.

        

$ npm version patch         # 1.0.x
$ npm version minor         # 1.x.0
$ npm version major         # x.0.0

If we updated the NPM package with a minor patch, the initial version within our package.json file would change from 1.0.0 to 1.1.0.

Outdated packages

To check for outdated packages:

        

$ npm outdated

Let's create a project

Begin by opening your terminal and creating a new folder/directory named 'my-project':

        

$ mkdir my-project
$ cd my-project

Create a package.json file:

        

$ npm init -y

Install the webpack and webpack-cli packages as devDependencies using NPM:

        

$ npm install webpack webpack-cli --save-dev

Install react and the react-dom library as dependencies using NPM:

        

$ npm install react react-dom --save

Once installed the NPM packages will be listed within the package.json file and a node_modules directory will be generated within your project folder. The package-lock.json file may also be generated and appear within the root of your project folder once you begin to install packages.
