JavaScript: Array

The JavaScript Array object is a high-level list-like object used to store multiple items of any data type. Arrays are created by using square brackets with individual elements separated by commas.

JavaScript Arrays are mutable, meaning that the values they contain can be changed. Even if the array is declared using const, the contents can be manipulated by reassigning internal values or using methods like push() and pop().

        

// An array containing numbers
const numberArray = [0, 1, 2, 3];

// An array containing different data types
const mixedArray = [1, "chicken", false, null];

// An array containing the same data type
const fruits = ["apple", "orange", "banana"];

Accessing Array elements

Elements within an array are arranged by index values, starting at 0 as the first element index. Elements can be accessed by their index using the array name and the index surrounded by square brackets. Array items can be changed by accessing the item and assigning it to a new value.

        

// An array containing numbers
const numberArray = [100, 200, 300, 400];

// Access the second element in the array
numberArray[1];     // 200

// Changing a value at an index
numberArray[1] = 205

console.log(numberArray);   // [100, 205, 300, 400]

indexOf

indexOf is used to return the first index at which a given element can be found in an array. It is widely used to check whether an element is in an array or not.

        

const sports = ["football", "hocky", "judo"];

const judoIndex = sports.indexOf("judo");
console.log(judoIndex);     // 2

The length property

The length property of a JavaScript array indicates the number of elements that array contains. To obtain the length value, append the length property after the array name using a period.

        

const numberArray = [1, 2, 3, 4];

numberArray.length;     // 4

The length property can also be used to append a new element to an array.

        

const numberArray = [1, 2, 3, 4];

numberArray[numberArray.length] = 5;    // [1, 2, 3, 4, 5]

Loop through the array

We can use a for loop to loop through an array and test it against the length property. This will output each element listed within the array.

        

const numberArray = [1, 2, 3, 4];

// Loop through the length of the array
for (let i = 0; i < numberArray.length; i++) {
    console.log(numberArray[i]);
}

The variable may be best declared outside of the loop to avoid it being declared on each iteration of the loop.

        

const numberArray = [1, 2, 3, 4];
// Declare the variable outside the loop
let i;

// Loop through the length of the array
for (i = 0; i < numberArray.length; i++) {
    console.log(numberArray[i]);
}

The push() method

The push() method of the JavaScript array can be used to add one or more elements to the end of an array.

        

const cart = ["apple", "orange"];

// Adding a single element to the array
cart.push("banana");

console.log(cart);      // ["apple", "orange", "banana"]

// Adding multiple elements to the array
cart.push("pear", "strawberry", "mango");

console.log(cart);      // ["apple", "orange", "banana", "pear", "strawberry", "mango"]

The pop() method

The pop() method removes the last element from an array and returns that element.

        

const cart = ["apple", "orange"];

// Removing the last element from the array
cart.pop();

console.log(cart);      // ["apple"]

reverse()

The reverse method is used to reverse the array.

        

const numbers = [1, 2, 3, 4, 5];

numbers.reverse();
console.log(numbers);   // [5, 4, 3, 2, 1]

The concat() method

The concat() method creates a new array by merging (concatenating) multiple arrays. The concat() method does not change the existing arrays, it always returns a new array.

        

const italianCars = ["Ferrari", "Pagani", "Lamborghini"];
const germanCars = ["BMW", "Audi", "Mercedes", "VW"];
const myGarage = italianCars.concat(germanCars);    // Joins italianCars and the germanCars arrays

console.log(myGarage);      // ["Ferrari", "Pagani", "Lamborghini", "BMW", "Audi", "Mercedes", "VW"]
