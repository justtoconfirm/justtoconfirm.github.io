Flexbox

CSS Flexible Box Layout, commonly known as Flexbox, is a CSS3 web layout model. The flex layout allows responsive elements within a container to be automatically arranged depending upon screen size.

The flexible box layout is a way of using the display property of CSS in a whole new manner. This provides more control to developers who want to build naturally responsive layouts.

Browser support

Flexbox is supported in a number of browsers. Browser support can also be confirmed using caniuse.

How to use Flexbox

The display: flex container may have a fixed or fluid width, but it makes sense to leave it fluid so the content adapts with the page. The internal elements are not given specific width values but instead, display based on proportions.

Flexbox is often considered as being one dimensional as it deals with layout in one dimension at a time - either as a row or as a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

The flex value is an alternative to block elements floated and manipulated using media queries.

We use a CSS property named flex to define a ratio between boxes - usuall HTML <div> elements with flex: 1 representing a 1:1 ratio where each internal flex item displays at the same width regardless of screen resolution.

The display property

As discussed, the first important part of a flexbox is the container itself. This container is given display: flex so its children behave in the expected manner.

        

<div class="container">
    ...
</div>
        
    
        

.container {
    display: flex;
}

Flex properties

You will very rarely see the flex-grow, flex-shrink, and flex-basis properties used individually; instead they are combined into the flex shorthand. The flex shorthand allows you to set the three values in this order - flex-grow, flex-shrink, flex-basis.

        

<div class="container">
    <div class="container__item">One</div>
    <div class="container__item">Two</div>
    <div class="container__item">Three</div>
    <div class="container__item">Four</div>
</div>
        
    
        

.container {
    display: flex;
}

.container__item {
    flex: 1 1 auto;     // flex-grow, flex-shrink, flex-basis
}

The flex-wrap property

The flex-wrap property specifies that the flex items will wrap if necessary. This is added to the container.

        

.container {
    display: flex;
    flex-wrap: wrap;
}
