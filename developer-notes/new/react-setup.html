React: Setup

Install React

NPM (Node Package Manager) can be used to install React. Both the react and react-dom NPM packages will be required to be installed. Open your terminal and install both packages as dependencies within the package.json file.

        

$ npm install react react-dom --save

Here are the packages that are being installed:

  • react - the top-level API of the React library
  • react-dom - DOM specific methods for React

These steps require Webpack to be installed and setup. Please read the Webpack setup documentation before installing React.

You may want to also install the @babel/preset-react package using NPM as a devDependency to allow components to be written in the ES6 JavaScript syntax within React.

        

$ npm install @babel/preset-react --save-dev

When the @babel/preset-react package is installed, the .babelrc config file will also need to be updated. Update the .babelrc file so it contains the following code:

        

{
    "presets": [ 
        "@babel/preset-env",
        "@babel/preset-react" 
    ]
}

Babel is a JavaScript compiler that allows you to use the ES6 JavaScript syntax in your code.

Creating the HTML document

Following the initial setup a simple HTML document is required that will allow the React application to render. Create a new HTML document named index.html within the distribution (dist) directory of your project. The HTML code should appear as below:

        

<!DOCTYPE html>
<html lang="en">
<head>
    <title>React Application</title>
</head>
<body>

<!-- Root element -->
<div id="app"></div>

<script src="main.js"></script>

</body>
</html>

Within the HTML document, a root element is added with the id attribute of "app". This is where the React application will be rendered.

The root element is managed by the ReactDOM library and is used to render the React application within HTML. This can be named anything you'd like, but for simplicity this has been named "app".

The main.js file that was created within the Webpack setup documentation is referenced within the <script> tag within this HTML code example.

Introduction to React

The smallest React example we can create is to display "Hello World" onto our HTML document. First, the React and ReactDOM libraries need to be imported.

Open the index.js file found in the src directory, remove any previous JavaScript code and update with the following import statements:

        

import React from 'react';
import ReactDOM from 'react-dom';

Once the two libraries have been imported, we can use the render method provided by the ReactDOM library to render "Hello World" onto the page. Your index.js file should now look like this:

        

import React from 'react';
import ReactDOM from 'react-dom';

ReactDOM.render(
    <h1>Hello World</h1>,
    document.getElementById('app')
);

Alternatively, you can import just the render method from the ReactDOM library by amending the import statement like so to reduce the need to import all of the ReactDOM library:

        

import React from 'react';
import { render } from 'react-dom';

You will need to use Webpack to rebuild your application for each change you make to your React application. Once complete, refresh your HTML webpage in the browser and "Hello World" should be displayed on the page.

        

$ npm run build

Run Webpack to rebuild your application for each change you make to your React application and refresh the browser to see your changes.
