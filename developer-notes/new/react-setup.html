React: Setup

Install React

NPM (Node Package Manager) can be used to install React. Both the react and react-dom NPM packages will be required to be installed. Open your terminal and install both packages as dependencies within the package.json file.

        

$ npm install react react-dom --save-dev

Here are the packages that are being installed:

  • react - the top-level API of the React library
  • react-dom - DOM specific methods for React

These steps require Webpack to be installed and setup. Please read the Webpack setup documentation before installing React.

You may want to also install the @babel/preset-react package using NPM as a devDependency to allow components to be written in the ES6 JavaScript syntax within React.

        

$ npm install @babel/preset-react --save-dev

When the @babel/preset-react package is installed, the .babelrc config file will also need to be updated. Update the .babelrc file so it contains the following code:

        

{
    "presets": [ 
        "@babel/preset-env",
        "@babel/preset-react" 
    ]
}

Babel is a JavaScript compiler that allows you to use the ES6 JavaScript syntax in your code.

Creating the HTML document

Following the initial setup a simple HTML document is required that will allow the React application to render. Create a new HTML document named index.html within the distribution (dist) directory of your project. The HTML code should appear as below:

        

<!DOCTYPE html>
<html lang="en">
<head>
    <title>React Application</title>
</head>
<body>

<script src="main.js"></script>

</body>
</html>

The main.js file that was created within the Webpack setup documentation is referenced within the <script> tag within this HTML code.
