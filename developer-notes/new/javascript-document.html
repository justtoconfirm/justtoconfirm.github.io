JavaScript: Document Object

The topmost object in the Document Object Model (DOM) is the document object. It represents the web page loaded into the current browser window or tab. The DOM is vital to accessing and amending the contents of the current web page.

Properties

Here are some properties of the document object.

Property Description
document.title Title of the current document
document.lastModified Date on which the document was last modified
document.URL Returns a string containing URL of current document
document.domain Returns domain of current document

Methods

The following are a few of the methods that select content or update content on the page.

Property Description
document.write() Writes text to document
document.getElementById() Returns element, if there is an element with the value of the id attribute that matches
document.querySelectorAll() Returns list of elements that match a CSS selector which is specified as a parameter
document.createElement() Creates a new element
document.createTextNode() Creates new text node

We can use these properries and methods to output details about the page using the document object.

        

var msg = '<p>page title: ' + document.title + '<br>';
msg += 'page address: ' + document.URL + '<br>';
msg += 'last modified: ' + document.lastModified + '</p>';

var el = document.getElementById('footer');
el.innerHTML = msg;

The URL will look very different if you run this page locally rather than on a web server.
