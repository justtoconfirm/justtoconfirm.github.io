JavaScript: Date

The Date object

JavaScript comes with a built-in Date object and related methods. The Date object is a built-in object in JavaScript that stores the date and time. It provides a number of built-in methods for formatting and managing the data.

To demonstrate the JavaScript Date object, a variable can be created and the current date can be assigned to it. Today's date is the 7th October, so this will be the current date, time and timezone displayed.

        

// Set a variable to the current date and time
const now = new Date()

// View the output
now

This will output in the console: Mon Oct 07 2019 08:41:00 GMT+0100 (British Summer Time)

Here we are creating a new instance of the Date object and assigning it to a variable named 'now'.

JavaScript understands the date based on a timestamp derived from Unix time which is a value consisting of the number of milliseconds that have passed since midnight on January 1st, 1970. We can get this timestamp with the getTime() method.

This timestamp is basically the milliseconds since Epoch time.

        

// View the output
now.getTime()

This number that is output for the current timestamp represents the same value as the above, 7th October, 2019.

Epoch time was chosen as a standard for computers to measure time inn earlier days of programming and it's the method that JavaScript uses.
