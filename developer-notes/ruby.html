Ruby

Ruby is a dynamic, open source programming language. It was created by Yukihiro Matsumoto, or "Matz", in Japan in the mid 1990's. Ruby is used today in web application development. The Ruby on Rails framework, built with the Ruby language by David Heinemeier Hansson, has since been introduced.

Ruby files usually have the file extension of '.rb', so for example: hello-world.rb

To run a Ruby file from within the terminal, use:

        

$ ruby hello-world.rb

If you're using MacOS, open your terminal and enter:

        

$ irb

This will open the IRB (Interactive Ruby).

Ruby Version Manager (RVM)

To check the version of Ruby installed:

        

$ ruby --version

On Mac OS, Ruby is installed by default. To manage Ruby versions, you can use the Ruby Version Manager (RVM). In the terminal, install RVM: 

        

$ curl -L https://get.rvm.io | bash -s stable

Once this command has finished running, you may need to restart your terminal for RVM to be recognised. To use RVM to install a specific version of Ruby, use:

        

$ rvm install ruby-[version]

// Example
$ rvm install ruby-2.4.1

If you want to set the latest version of Ruby as the default version, in the terminal type:

        

$ rvm --default use 2.4.1

Don't forget to replace 2.4.1 with the version number of Ruby you want to use.

Sass

To install Sass using Ruby, we need to install it as a gem. This will install libsass with Ruby as Ruby Sass is being deprecated at the end of March 2019.

        

$ gem install sassc

// Mac OS
$ sudo gem install sassc

Ruby Gems

Ruby Gems allow you to easily download, install and use packages on your system. This package is called a "gem" which contains a package Ruby application or library. Gems can be used to extend or modify functionality in your Ruby applications. Some Gems will provide command line utilities to help automate tasks.
