jQuery

jQuery is a lightweight JavaScript library. It takes a lot of common tasks that require many lines of JavaScript code, and wraps them into methods that you can call with a single line of code. jQuery also simplifies a lot of advanced features of JavaScript, like AJAX calls and DOM manipulation. In addition, jQuery plugins are available for additional functionality.

With jQuery you select (query) HTML elements and perform 'actions' on them.

Setup

jQuery can be downloaded from the official jQuery website.

There are two versions of jQuery available:

  • Production - this is for you web page/website, its been minified and compressed.
  • Development - this is for testing and development, its uncompressed and readable code.

The jQuery library is a single JavaScript file that is referenced in your HTML. I usually download the jQuery library and place the <script> tag before the closing <body> element within the HTML code to include the library.

        

<script src="js/jquery.js"></script>

</body>

Any jQuery plugins that are then used need to be included below the jQuery script element.

        

<script src="js/jquery.js"></script>
<script src="js/jquery-plugin.js"></script>

</body>

You may also see when using the production version of the library, the jQuery includes the version number and has a slightly different file name.

        

<script src="js/jquery-3.3.1.min.js"></script>
<script src="js/jquery-plugin.min.js"></script>

</body>

If you don't want to download and host jQuery yourself, you can include it from a CDN (Content Delivery Network). Both Google and Microsoft host jQuery. Examples can be found on https://code.jquery.com/.

        

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

Document Ready

jQuery methods are often placed inside a document ready event. This is to prevent any jQuery code from running before the document is finished loading (is ready). It is good practice to wait for the document to be fully loaded and ready before working with it. This also allows you to have your JavaScript code before the <body> element of your document, in the <head> section.

        

$(document).ready(function() {
    // jQuery methods go here ...
});

The jQuery team have also created a shorter method for the document ready event:

        

$(function() {
    // jQuery methods go here ...
});

Here are some examples of actions that can fail if methods are run before the document is fully loaded:

  • Trying to hide an element that is not yet created.
  • Trying to get the size of an image that is not yet loaded.

The jQuery Syntax

The jQuery syntax is ideal for selecting HTML elements and performing some action on the element(s).

  • The $ sign to define/access the jQuery library.
  • A (selector) to "query" (find) HTML element(s).
  • A jQuery action() to be performed on the element(s).

Here are a few examples:

        

$(this).hide();         // Hides the current element

$("p").hide();          // Hides all <p> elements

$(".myclass").hide();   // Hides all elements with the class 'myclass'

$("#mytest").hide();    // Hides the element with the id of 'mytest'

jQuery Selectors

jQuery selectors allow you to select and manipulate HTML element(s). They are used to "find" (select) HTML elements based on their name, id, class, type, or attributes. All selectors in jQuery start with the dollar sign ($) followed by parenthesis.

        

$()

The jQuery element selector selects elements based on the name of the element. You can select all <p> elements on a page like this:

        

$("p")

In this example, when a user clicks on a button, all <p> elements will be hidden:

        

$(document).ready(function() {
    $("button").click(function() {
        $("p").hide();
    });
});

jQuery Events

jQuery is tailor-made to respond to events in an HTML page. The actions that a web page can respond to are called events. An event represents the precise moment when something happens. Examples include:

  • Moving the mouse cursor over an element.
  • Selecting a radio button.
  • Clicking on an element.

In jQuery, most DOM events have an equivalent jQuery method. To assign a click event to all paragraphs on a page, you can do this:

        

$("p").click();

The next step is to define what should happen when the event fires. You must pass a function to the event.

        

$("p").click(function() {
    // An action goes here ...
});

jQuery noConflict()

jQuery uses the dollar sign ($) as a shortcut for jQuery. There are many other JavaScript frameworks and libraries that may also use the $ sign as a shortcut. If two or more frameworks and libraries are using the same shortcut, one of them may stop working. the jQuery team have implemented the noConflict() method to prevent this.

This method releases the hold on the $ shortcut that jQuery has, so other scripts and libraries/frameworks can use it. This still allows jQuery to be used by writing the full name of 'jQuery' instead of the shortcut ($).

        

$.noConflict();
jQuery(document).ready(function() {
    jQuery("button").click(function() {
        jQuery("p").hide();
    });
});

Your own shortcut can also be created:

        

var jq = $.noConflict();
jq(document).ready(function() {
    jq("button").click(function() {
        jq("p").hide();
    });
});

If you have a block of code that you don't want to change, you can pass the $ sign in as a parameter to the ready method. This allows you access to jQuery inside the function. Outside the function, you would still need to use 'jQuery'.

        

$.noConflict();
jQuery(document).ready(function($) {
    $("button").click(function() {
        $("p").hide();
    });
});
