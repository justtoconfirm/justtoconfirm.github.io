CSS Animation

CSS animations allow animation of most HTML elements without using JavaScript and make it possible to transition from one CSS style configuration to another. Animations consist of two components, a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animation's style.

To create a CSS animation sequence, you style the element you want to animate with the animation property. This lets you configure the timing, duration, and other details of how the animation sequence should progress.

The actual appearance of the animation is done using the @keyframes at-rule.

This simple example styles the <div> element so that the element slides in from off the right edge of the browser window. Animations like this can sometimes cause the page to become wider than the browser window. To avoid this problem, put the element to be animated inside a container and set overflow: hidden on the container.

We can begin by creating the HTML <div> element:

        

<div class="box box--slide">I can slide</div>

Next, we add the CSS to provide the animation.

        

.box {
    background-color: #3d7e9a;
    color: #fff;
}

.box--slide {
    animation-duration: 3s;
    animation-name: slidein;
}

@keyframes slidein {
    from {
        margin-left: 100%;
        width: 300%; 
    }

    to {
        margin-left: 0%;
        width: 100%;
    }
}

Example

I can slide

Should we want to animation to repeat itself, simply use the animation-iteration-count property to indicate how many times to repeat the animation. In this case, let's use infinite to have the animation repeat indefinitely.

        

.box--slide {
    animation-duration: 3s;
    animation-name: slidein;
    animation-iteration-count: infinite;
}

Having made it repeat, it's very odd to have it jump back to the start each time it begins the animation. What we really want is for it to move back and forth across the screen. This can be accomplished by setting the animation-direction property to alternate.

        

.box--slide {
    animation-duration: 3s;
    animation-name: slidein;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

Of course we can also use shorthand for this:

        

.box--slide {
    animation: 3s infinite alternative slidein;
}
