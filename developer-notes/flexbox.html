Flexbox

CSS has always been used to design the layout and look of web pages by producing complicated multi-column layouts. First we used tables, floats, positioning, and inline-block. All these techniques left out major functionality, such as vertical centering. To solve these issues, new responsive layout models became available natively in the browser. Flexbox and CSS Grid are two examples of these models that are widely supported across multiple browsers. These not only allow us to create layouts that were only possible previously when using JavaScript, but also help to make the code easier to maintain and understand.

The Flexible Box Module, usually referred to as flexbox, is a one-dimensional layout modal. Flexbox makes it easier to design flexible responsive layout structures without using float or positioning. The flexbox properties are supported in all modern browsers (Internet Explorer 11 and above). When we describe flexbox as being one dimensional we are describing the fact that flexbox deals with layout in one dimension at a time - either as a row or as a column. This can be contrasted with the two-dimensional model of the CSS Grid Layout, which controls columns and rows together.

You can check the browser support for Flexbox using caniuse.com.

Flexbox v CSS Grid

The core difference between CSS Grid and Flexbox, is the CSS Grid's approach is layout-first, while Flexbox' approach is content-first. If you only need to define a layout as a row or column, then Flexbox is probably the solution you need. If you want to define a grid and fit content into it in two dimensions, the CSS Grid is the preferred method.

Parent Element - Container

An area of a document laid out using flexbox is called a flex container. To create a flex container, we set the value of the area's container's display property to flex or inline-flex. As soon as we do this the children of that container become flex items. The result of this is that your items will all line up in a row, using the size of the content as their size in the main axis.

We start by defining a parent element, the container. This container (flex container), becomes flexible by setting the display property to flex. We need to write the HTML to begin.

        

<div class="wrapper">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
</div>

Once the HTML has been created, we can then set the CSS property.

        

.wrapper {
    display: flex;
}

When working with Flexbox, you need to think in terms of two axes, the main axis and the cross axis. The main axis is defined by the flex-direction property. The cross axis runs perpendicular to it. Everything we do with Flexbox refers back to these axis.

The main axis is defined by the flex-direction property. This has four possible values:

  • row
  • row-reverse
  • column
  • column-reverse

Should you choose row or row-reverse, your main axis will run along the row in the inline direction.

        

.wrapper {
    display: flex;
    flex-direction: row;
}

The cross axis runs perpendicular to the main axis, therefore if your flex-direction (main axis) is set to row or row-reverse, the cross axis tuns down the columns. If your main axis is column or column-reverse, then the cross axis runs along the rows.

Flex Wrap

If there are more items that can fit in the container, they will not wrap but will instead overflow. To cause wrapping behaviour, add the flex-wrap property with the value of wrap. Set it to nowrap, which is also the initial value, and they will instead shrink to fit the container. Using nowrap would cause an overflow if the items were not able to shrink, or could not shrink small enough to fit.

The flex-wrap property with the wrap value specifies that the flex items will wrap if necessary.

        

.wrapper {
    display: flex;
    flex-wrap: wrap;
}

Align Content

The justify-content property is used to align the flex items.

        

.wrapper {
    display: flex;
    justify-content: center;
}

If you want to align the flex items at the end of the container (right), then use the flex-end value. to align at the start of the flex container (left and the default value), use flex-start.

        

// Align left (default)
.wrapper {
    display: flex;
    justify-content: flex-start;
}

// Align right
.wrapper {
    display: flex;
    justify-content: flex-end;
}
