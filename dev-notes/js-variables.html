Variables

A variable is a container for storing a value, like numbers we can use in a calculation, or a string we might use within a sentence.

Declaring a variable

To use a variable you've first got to create it, or more accurately, we call this declaring the variable. To do this, we use the keyword var followed by the name you want to call the variable. The name of the variable is often called the identifier.

Here we are creating two variables called myName and myAge. The variables declared have no value; they are empty containers. A variable without any value is an undefined variable. When you enter the variable names in the console, you should get a value of undefined returned.

If the variable does not exist, you'll get an error message.

                        

var myName;
var myAge;

To assign a value to a variable - also known as initializing a value - we use the assignment operator (=). We will not assign values to the two variables declared. When the variable name is used within the console, the value will be returned.

Initializing variables

We are declaring and initializing the variables at the same time.

                        

var myName = 'Dan';
var myAge = 30;

Updating and changing values

Once a variable has been initialized with a value, you can change (or update) the value by providing it a different value. Let's change the value of myName to 'Bob':

                        

// Declare and initialize the variable
var myName = 'Dan';
// Change the value
myName = 'Bob';

Unlike some other languages, you don't need to specify what data type (e.g. numbers, strings, arrays, etc) a variable will contain. This is because JavaScript is a 'loosely types language'. For example, if you declare a variable and give it a value within quotes, the browser will treat the variable as a string.

                        

// A String
var myString = 'Hello';
// Also a String
var myNumber = '500';
// Not a String, but a Number
var myOtherNumber = 500;

We can use the special function called typeof() to return the data type of the variable that you pass into it.

                        

var myString = 'Hello';
typeof(myString);     // returns: 'string'
var myNumber = '500';
typeof(myNumber);     // returns: 'string'
var myOtherNumber = 500;
typeof(myOtherNumber);    // returns: 'number'

A simple example using variables to calculate the total value:

                        

var num1 = 24;
var num2 = 17;
var total = num1 + num2;
total;    // 41

You can declare variables using the keywords of: var, let or const. As long as a variable is not declared as a constant - using the const keyword - you can change it's value. A variable declared as a constant should have it's name in uppercase. This is a general convention.

                        

// Declare a variable using the var keyword
var myName = 'Dan';
// Declare a variable using the let keyword
let myName = 'Dan';
// Declare a variable as a constant
const PI = 3.14;
