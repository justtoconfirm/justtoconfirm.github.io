Python

Python is a high-level, open-source, dynamically typed programming language. Two major versions of Python are currently in active use:

  • Python 3.x is the current version and is under active development.
  • Python 2.x is the legacy version and will receive only security updates until 2020. No new features will be implemented.

A Python file is usually created with the file extension of .py

To verify the version of Python installed, you can use the following terminal command:

		

$ python --version

The $ symbol does not need to be entered into the terminal. This symbol simply indicates that this is a terminal command.

If you have installed Python 3 but the output shows as Python 2, you may have both versions installed. To explicitly use the Python 3 interpreter:

		

$ python3

Variables

To create a variable in Python you will need to specify the variable name and assign a value to it. Python uses = to assign a value to a variable. There is no need to declare or assign a data type to a variable in Python. Assigning a value to a variable itself, declares and initializes the variable with that value.

The Python interpreter will specify the most suitable data type for each variable when they are declared. There is no way to declare a variable without assigning an initial value to it.

			

a = 2
b = 3
c = a + b

print(c)		# 5

In Python, the semi-colon ; is not needed at the end of each statement.

We can check the datatype of an object using the built-in type() function.

			

a = 2
print(type(a))			# <type 'int'>

name = "Dan"
print(type(name))		# <type 'str'>

Datatypes can be conveted from one type to another. For example, "123" is a string, but can be converted to an integer using the int() function.

			

a = "123"
b = int(a)				# convert String to int

print(type(b))		# <type 'int'>

In Python everything is an object. When we define a variable, we are creating an object and giving it a name so we can use it later. For example, below, we are creating an object of 5 and giving it the name of a.

			

a = 5

If we then change the object to 6:

			

a = 6

We are not modifying the object, but rather we are creating a new object of 6 and giving it a name of a. In Python we can never modify an int object, but rather we create a new object and give it the same name. This is an immutable data type.

Mutable data types are those in which we can modify the object.

Immutable Data Types

  • int, long, float, complex
  • str
  • byte
  • tuple

Mutable Data Types

  • bytearray
  • list
  • set
  • dict

Lists

The list type is probably the most commonly used collection type in Python. A list is similar to an array in many other languages. In Python, a list is an ordered collection of values.

			

int_list = [ 1, 2, 3 ]
string_list = [ 'abc', 'def' ]
empty_list = []

The elements of a list are not restricted to a single data type.

			

mixed_list = [ 1, 'abc', True, 2.34, 'def' ]

The elements inside of a list can be accessed using an index (numerical representation of their position). The first element in the list has an index of 0, the second element has an index of 1, and so on.

			

names = [ 'Dan', 'Alice', 'Bob' ]

print(names[0])					# Dan
print(names[1])					# Alice

Indexes can also be negative values to count from the end of the list. The last element in a list will have the index of -1.

			

print(names[-1])				# Bob

Lists are mutable meaning the values can be changed within the list.

			

# Change 'Alice' to 'Cassie'
names[1] = "Cassie"
print(names)						# ['Dan', 'Cassie', 'Bob']

To add a new element onto the end of the list:

			

# Add 'Fred' to the list of names
names.append("Fred")
print(names)						# ['Dan', 'Cassie', 'Bob', 'Fred']

To get the index of a value in the list:

			

names.index("Cassie")		# 1

The length (number of items in the list) can be output:

			

len(names)							# 4

We can also count the number of occurences of any item in the list:

			

a = [ 1, 1, 1, 1, 2, 3, 4 ]
a.count(1)							# There are 4 number '1' values

Tuples

A tuple is similar to a list except that it is fixed-length and cannot be changed (immutable). No values can be added or removed from the tuple. Tuples are commonly used for small collections of values that do not need to changes, such as IP addresses and ports. Tuples are represented with parenthesis instead of square brackets.

			

ip_address = ( '127.0.0.1', 8080 )

Dictionary

A dictionary is a collection of key-value pairs. It is surrounded by curly braces. Each pair is separated by a comma and the key-value pairs are separated by a colon.

			

city = {

	'UK': 'London',
	'USA': 'Wasington DC',
	'France': 'Paris'

}

# To get a value, refer to it's key
uk_capital = city[ 'UK' ]
print(uk_capital)						# London

Set

A set is a collection of elements with no repeats and without an insertion order, but has a sorted order. They are used for situations where it's only important that some things are grouped together, and not the order they were included. For large groups of data, it is much faster to check whether or not an element is in a set that to do the same for a list.

			

first_names = { 'Adam', 'Beth', 'Charlie' }

You can build a set using an existing list:

			

my_list = [ 1, 2, 3 ]
my_set = set(my_list)

Check membership of the set using in:

			

if name in first_names:
	print(name)

Module

A module is an importable file containing definitions and statements. A module can be created by creating a .py file. Functions in a module can be used by importing the module into a file where the function will be used.

			

def say_hello():
	print("Hello World")

For modules you have created, they will need to be in the same directory as the file you are importing them into.

			

import hello 						# Import hello.py
hello.say_hello()				# 'Hello World'

Specific functions of a module can be imported:

			

from hello import say_hello
say_hello()							# 'Hello World'

Modules can also be given an alias:

			

import hello as xy
xy.say_hello()					# 'Hello World'

Modules can also be turned into stand-along runnable scripts.

			

if __name__ == '__main__':
	import hello

We can run stand-alone scripts using the following command:

		

$ python run_hello.py 		# 'Hello World'

External Modules

pip is used to install any packages available at the python package index (PyPI). Pip is installed by default if you're using Python 2 (>= 2.7.9) or Python 3 (>= 3.4). pip often refers to Python 2 and pip3 to Python3.

Using pip will only install packages for Python 2 and pip3 will only install packages for Python 3.

To import a module, use the import statement. import module will import a module and then allow you to reference its objects - values, functions, and classes - for example, using the module.name syntax.

In the example below, the 'random' module is imported which contains the randint function. So by importing random, you can call randint with random.randint.

			

import random
print(random.randint(1, 10))

Using the import statement will provide you with a namespace.

Functions

Functions in Python provide organised, reusable code to perform a set of actions. To declare a function, Python uses the def keyword.

			

def function_name():
	# Statement here

def function_name( parameters ):
	# Statement here

def greet():
	print("Hello")

The function_name is known as the identifier of the function. Statements are also known as the function body and are executed each time the function is called.

We can call a function, like the example below:

			

greet()							# Hello

We can create a function that will take one single argument and display the passed-in value each time the function is called:

			

def greet( name ):
	print(name)

greet("Dan")				# 'Dan'

A function that reaches the end of the execution without a return statement will always return: None

Virtual Environments

A Virtual Environment is a tool to keep the dependencies required by different projects in separate places. virtualenv is a tool to build isolated Python environments. This program creates a folder which contains all the necessary executables to use the packages that a Python project would need.

Classes

Python is a scripting language that also supports object-oriented programming. Classes describe data and provide methods to manipulate that data, contained within a single object. Code using classes is generally easier to read, understand, and maintain.

A class is a template that defines the basic characteristics of a particular object.
