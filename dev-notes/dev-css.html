CSS

CSS (Cascading Style Sheets) is a language used to describe the style of HTML documents and describes how HTML elements should be displayed.

CSS files have the file extension of .css

A CSS rule-set consists of a selector and declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and property value, separated by a colon.

				

selector {
	/* Declaration block */
}

/* Example */
h1 {
	color: #f04124;
	font-size: 12px;
}

CSS selectors are used to select HTML elements based on their element name, id, class or attribute.

As a simple example, lets use the p selector to find all <p> paragraph elements with the HTML page and set the text colour:

				

p {
	color: #f04124;
}

HTML code:

				

<html>
<head>

	<link rel="stylesheet" href="styles/stylesheet.css" />

</head>
<body>

	<p>This text will be red</p>

</body>
</html>

The class selector selects elements with a specific class attribute. To select elements with a specific class, write a period . followed by the name of the class.

				

p {
	color: #f04124;
}

.new-paragraph {
	color: #1779ba;
}

Lets update the HTML code:

				

<p>This text will be red</p>
<p>This text will be red</p>
<p class="new-paragraph">This text will be blue</p>

A class should be used if you intend to re-use it more than once and/or on multiple elements. An id selector should only be used once and should be unique.

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element should always be unique within a page.

To select an element with an id, write a hash # followed by the id of the element.

				

/* Element */
p {
	color: #f04124;
}

/* Class */
.new-paragraph {
	color: #1779ba;
}

/* id */
#paraOne {
	background-color: #4caf50;
}

Now, lets update the HTML to apply the id (we do not need to include the hash when we use the id within the HTML code):

				

<p>This text will be red</p>
<p>This text will be red</p>

<!-- We can use the class as many times as we wish -->
<p class="new-paragraph">This text will be blue</p>
<p class="new-paragraph">This text will be blue</p>

<!-- This id is unique so can we can only use the id once in the HTML -->
<p class="new-paragraph" id="paraOne">This text will be blue with a background colour</p>

CSS & HTML

Usually CSS is placed in an external stylesheet so the code can be shared between multiple HTML pages. This allows changes to be made in a single CSS file to change the styling of an entire website very quickly. Each page that requires the stylesheet must include a reference to the CSS within the <head> section of the HTML code.

Find further notes on HTML here.

This example is showing that the stylesheet is named stylesheet.css and is located with a styles folder/directory.

				

<head>

<link rel="stylesheet" href="styles/stylesheet.css" />

</head>
