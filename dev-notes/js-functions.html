Functions

We can use functions to section off small bits of logic or tasks that we can then call when we need to. Functions can help take large problems and break them into small manageable chunks.

Declaring a function

To create a function it must first be declared using the function keyword. Declaring the function gives the function a name and specifies what the function is to do when it is called.

                        

function myFunc () {
 
   // Code is added here
 
}

Calling a function

Defining a function does not execute it. The function needs to be called.

                        

function myFunc () {
 
   // Code is added here
 
}
 
// Call the function
myFunc();

Parameters & Arguments

Below is an example of a simple function that takes two numbers as arguments and passes them into the num1 and num2 parameters. These values are then multiplied together, the result is stored in a variable and returned using a return statement.

When we call the function, the two values we want to multiply are passed as arguments into the function.

                        

function multiplyNumbers ( num1, num2 ) {
 
   var result = num1 * num2;
   return result;
 
}
 
// Call the function to multiply 4 and 2
multiplyNumbers(4, 2);

The return statement tells the browser to return the variable value so it can be used outside of the function. This is necesssary because a variable defined inside a function is only available inside of that function.

Return values are returned by the function once it executes. This is called variable scoping.

Anonymous Functions

An anonymous function is a function that has no name - it is anonymous.

If you're writing a one-off chunk of JavaScript that never needs to be referred elsewhere, it's wise to use an anonymous function.

The empty brackets after the function causes the function to automatically be invoked (called) on the page load. This is known as an immediately-invoked function expresssion (IIFE).

                        

( function () {
 
   // Code is added here
 
})();
