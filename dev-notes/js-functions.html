Declaring a function

To create a function it must first be declared using the function keyword. Declaring the function gives the function a name and specifies what the function is to do when it is called.

    	
    
    function myFunc () {
     
       // Code is added here
     
    }

Calling a function

Defining a function does not execute it. To call the function, follow the example code below:

    	
    
    myFunc();

We can now declare a function that will display an alert message when the function is called.

    	
    
    function helloAlert () {
     
       // Show alert when the function is called
       alert('Hello World');
     
    }
     
    // Call the function
    helloAlert();
