PHP

PHP (a recursive acronym for PHP: Hypertext Preprocessor) is an open-source programming language especially suited to web development.

A PHP file is created with the file extension of .php

PHP files placed in the web directory of a web server will automatically be parsed. PHP is executed by a web server with the resulting HTML send to the browser.

To check the version of PHP being used, open your terminal and enter the following command:

			

$ php --version

The $ symbol does not need to be entered into the terminal. This symbol simply indicates that this is a terminal command.

To display all the information about the PHP configuration.

				

<?php

	phpinfo();

	phpinfo(INFO_MODULES);		// Show just the module information

PHP Server

PHP vesions greater than 5.4 come with a built-in development server. It can be used to run applications without having to install a production HTTP server such as Nginx or Apache.

The built-in server is only designed to be used for development and/or testing purposes.

			

$ php -S <host/ip>:<port> -t <directory>

By default, this command will start the web server to listen on the port provided using the current directory, unless this is specified with the -t flag. If, for example, you have a php directory on the desktop and you want to use port 8080 for your PHP server, you can use the following command:

			

$ php -S localhost:8080 -t desktop/php

Each time a request is made from the development server, a log entry is written into the command line. To view the PHP code running, open the browser and enter the following URL:

			

http://localhost:8080

PHP CLI (Command Line Interface)

PHP can also be run directly from the command line using the CLI (Command Line Interface). To open the interactive PHP shell in the terminal:

			

$ php -a

PHP code can then be typed or pasted into the terminal and executed using the return key.

PHP Tags

The PHP parser looks for opening and closing PHP tags to get the code to be interpreted.

				

<?php

	...

?>

You should always use <?php and ?> tags. When a file is not expected to produce output (the entire file is written in PHP), the closing tag should be omitted to avoid unintentional output which can cause problems when a client parses the document.

Echo & Print

echo and print are language constructs, not functions. They do not require parenthesis around the argument like a function. They are used to output the string representation of a variable, constant or expression.

				

<?php

	// Assign the string 'Dan' to the variable $name
	$name = "Dan";

	// Output the value of $name using echo and print
	echo $name;
	print $name;

	// Parenthesis are not required, but can be used
	echo($name);
	print($name);

	// Output a human readable format of an array or object
	print_r();

You may have a variable that is an array or object. Trying to output this with echo will cause an error. You can instead use print_r() to dump a human readable format of this variable.

				

<?php

	$myArray = [ "Hello", "World" ];
	print_r($myArray);

Output:

			
			
Array {
	[0] => Hello
	[1] => World
}

We can output human readable debug information about the content of an argument, including it's type and value. Output information about the variable - including it's type - using the following code:

				

$myArray = [ "Hello", "World" ];
var_dump($myArray);

Output:

			
			
array(2) {
	[0]=>
  	string(5) "Hello"
  	[1]=>
  	string(5) "World"
}

Arrays

An array is a special variable that can hold more than one value at a time. An array in PHP is actually an ordered map where map is a type that associates values to keys. The first item in an array always has an index of 0, the second item has an index of 1 and so it continues.

An empty array can be initialized:

				

<?php

	$array = [];

Values can also be preset into an array.

				

<?php

	// Array preset with values
	$fruits = [ 'apple', 'pear', 'orange' ];

	// Get the second item in the array - pear - use the index of 1
	echo $fruits[1];

An array that is initialized with values and custom indexes, is called an associative array.

				

<?php

	$fruits = [
		'first' => 'apple',
		'second' => 'pear',
		'third' => 'orange'
	];

To get the second item in the associative array - pear - we reference the key of 'second':

				

<?php

	echo $fruits['second'];

Looping though an array can be done using a direct foreach loop.

				

<?php

	$fruits = [ 'apple', 'pear', 'orange' ];

	foreach ( $fruits as $fruit ) {

		echo "I am the fruit $fruit <br/>";

}

Output:

			

I am the fruit apple
I am the fruit pear
I am the fruit orange

We can also use a foreach loop for keys within an array.

				

<?php

	$fruits = [

		'first' => 'apple',
		'second' => 'pear',
		'third' => 'orange'

	];

	foreach ( $fruits as $key => $fruit ) {

		echo "Eating $fruit is $key <br/>";

	}

Output:

			

Eating apple is first 
Eating pear is second 
Eating orange is third

We can use the sort() function to sort the array into ascending order by value.

				

<?php

	$fruits = [ 'apple', 'pear', 'orange' ];
	sort($fruits);
	print_r($fruits);

Output:

			

Array 
( 
	[0] => apple 
	[1] => orange 
	[2] => pear 
)

The array can also be sorted into descending order by value using the rsort() function.

				

<?php
	
	$fruits = [ 'apple', 'pear', 'orange' ];
	rsort($fruits);
	print_r($fruits);

Output:

			

Array ( 
	[0] => pear 
	[1] => orange 
	[2] => apple 
)

PHP & MySQL

PHP 5 and later can work with a MySQL database using:

  • MySQLi (MySQL improved) extension
  • PDO (PHP Data Objects)

Earlier versions of PHP used the MySQL extension, but since 2012, this has now become deprecated.

PDO will work with 12 different database systems, MySQLi will only work with MySQL databases. Both are object-oriented, but MySQLi also offers a procedural API. When PHP 5 (or later versions) are installed, the MySQLi extension is automatically installed.

Before we can access a MySQL database, we need to connect to the server.

				

<?php
$servername = "localhost";
$username = "username";
$password = "password";

// Create a connection
$conn = new mysqli($servername, $username, $password);

// Check the connection
if ($conn->connect_error) {
	die("Connection failed: " . $conn->connect_error);
}
echo "Connection has been made";

The object-oriented example above uses $connect_error. This was broken until PHP 5.2.9 / 5.3.0. If you need to ensure compatibility with earlier PHP versions, use the following code:

				

// Check the connection
if (mysqli_connect_error()) {
	die("Database connection failed: " . mysqli_connect_error());
}

Object-Oriented PHP

Object-oriented programming is a style of coding that allows developers to group similar tasks into classes. This helps to keep code DRY (Don't repeat yourself) and easy to maintain. The major benefit is that if something changes, usually only one change is required to update the code rather than multiple changes.

OOP (Object-Oriented Programming) often introduces the concept of objects and classes. To begin, a class is declared using the class keyword followed by the name of the class.

				

<?php

class MyClass {
	
	// Class properties and methods go here ...

}

The class can then be instantiated and stored in a variable, using the new keyword.

				

<?php

class MyClass {
	
	// Class properties and methods go here ...

}

// Instantiated class to create a new Object
$obj = new MyClass;

To see the contents of the class, the var_dump() function can be used:

				

// Will output: object(MyClass)#1 (0) { }
var_dump($obj);

To add data to a class, properties (class-specific variables), are used. These are similar to regular variables, except they are bound to the object and can only be accessed using the object.

Multiple instances of a class can exist. If the individual object is not referenced, the script would be unable to determine which object to read from. The use of the arrow -> is an OOP construct used to access properties and methods of an object.

				

<?php

class MyClass {
	
	// This is a class property
	public $prop1 = "I am a class property";

}

// Instantiated class to create a new Object
$obj = new MyClass;

// Output the property
echo $obj->prop1;

If the browser is reloaded, it will output:

			

I am a class property

Methods are class-specific functions the object will be able to perform, that are defined within the class.

OOP allows objects to reference themselves using the $this keyword.

				

<?php

class MyClass {
	
	// This is a class property
	public $prop1 = "I am a class property";

	// This is a class method
	public function setProperty($newval) {

		$this->prop1 = $newval;

	}

	// This is a class method
	public function getProperty() {

		return $this->prop1 . "<br />";

	}

}

// Instantiated class to create a new Object
$obj = new MyClass;

// Get the property value - call the getProperty() method
echo $obj->getProperty();

// Set a new value - pass value as argument into setProperty() function
$obj->setProperty("I'm a new property value");

// Get the property value once again, should now be changed
echo $obj->getProperty();

// Output the property - because we can
echo $obj->prop1;

Multiple instances of the same class to create objects allows the properties and methods to be re-used.

				

<?php

class MyClass {
	
	// This is a class property
	public $prop1 = "I am a class property";

	// This is a class method
	public function setProperty($newval) {

		$this->prop1 = $newval;

	}

	// This is a class method
	public function getProperty() {

		return $this->prop1 . "<br />";

	}

}

// Create two objects
$obj = new MyClass;
$obj2 = new MyClass;

// Get the value of $prop1 from both objects
echo $obj->getProperty();
echo $obj2->getProperty();	

// Set new values for both objects
$obj->setProperty("A new property value");
$obj2->setProperty("I belong to the second instance");

// Output both objects' $prop1 value
echo $obj->getProperty();
echo $obj2->getProperty();

Constructor & Destructor

When an object is instantiated, it is often desirable to set a few things to begin with. To handle this, PHP provides the magic method of __construct(), which is called automatically whenever a new object is created. To call a function when the object is destoyed, the __destruct() magic method can be used. This is useful for class cleanup - such as closing a database connection, for instance. To trigger the destructor, you can destory the object using the unset() function.

				

unset($obj);

__CLASS__ returns the name of the class in which it is being called. This is known as a magic constant.

Here is a full code example:

				

<?php

class MyClass {
	
	// This is a class property
	public $prop1 = "I am a class property";

	// Constructor
	public function __construct() {

		echo 'The class "', __CLASS__, '" was initiated<br/>';

	}

	// Destructor
	public function __destruct() {

		echo 'The class "', __CLASS__, '" was destroyed.<br/>';

	}

	// This is a class method
	public function setProperty($newval) {

		$this->prop1 = $newval;

	}

	// This is a class method
	public function getProperty() {

		return $this->prop1 . "<br />";

	}

}

// Create a new object
$obj = new MyClass;

// Get the value of $prop1
echo $obj->getProperty();

// Destroy the object
unset($obj);

// Output a message at the end of the file
echo "End of file.<br />";
