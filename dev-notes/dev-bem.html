BEM

BEM - Block Element Modifier, is a methodology that helps you to create reusable components and code sharing in front-end development.

Block

A standalone entity that is meaningful on it's own. Any DOM (Document Object Model) element can be a block if it accepts a class name.

				

<div class="block"> ... </div>

To apply CSS to a block, the following rules should apply:

  • Use class name selector only.
  • No tag name or id selectors.
  • No dependency on other blocks or elements on the page.
				

.block {
	color: #ddeeff;
}

Element

An element must be tied to it's block. Within a block, all elements are semantically equal. When adding a CSS class to an element, the class name is formed from the block name, two underscores and the name of the element itself.

				

<!-- Block -->
<div class="block">

	<!-- Element -->
	<span class="block__elem"> ... </span>

</div>

The CSS should be then applied like so:

				

/* Block */
.block {
	color: #ddeeff;
}

/* Element */
.block__elem {
	color: #fafafa;
}

Modifier

A modifier is used to change the appearance, state, or how a block or element behaves. A CSS class is formed from a block or element's name, plus two dashes followed by the modifier itself. Spaces in the modifier name are replaced by a dash.

				

<!-- Block with modifier -->
<div class="block block--mod"> ... </div>

<!-- Block with two modifiers -->
<div class="block block--shadow block--size-small"> ... </div>

Add modifier classes only to blocks or elements they modify and keep the original class. In this example (above), the original class would be block.
