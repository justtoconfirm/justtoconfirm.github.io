Introduction to CSS

CSS is a language used to describe the style of an HTML document and describes how HTML elements should be displayed. CSS stands to Cascading Style Sheets and external stylesheets are stored in a CSS file with the file extension of .css

An external CSS stylesheet can be added within the <head> section of an HTML document.

    

    <link rel="stylesheet" href="stylesheet.css">

</head>

You may also see examples with the type attribute being used. Whilst this is valid code, in HTML5, the type attribute is no longer needed.

    

<link rel="stylesheet" type="text/css" href="stylesheet.css">

CSS Syntax

There are three main parts of CSS - selectors, properties and values.

  • Selectors - the element(s) being selected
  • Properties - the aspect of the element being styled
  • Values - the style being applied
    

selector {
    property: value;
}

A real world example:

    

h1 {
    color: blue;
}

Spacing between selectors is important. For example, say we have two selectors:

  • .foo.bar
  • .foo .bar

The first example targets a class of foo and a class of bar on the same element.

    

<div class="foo bar">This is an element</div>

The second example targets a class of bar that is a child of the foo class.

    

<div class="foo">
    <div class="bar">This is an element</div>
</div>

From least specific to most specific, here are the main types of selectors:

  • Universal selector - selects all elements
  • Type selector - targets element(s) by HTML tag (including pseudo elements)
  • Class & Attribute selectors - targets element(s) by class or attribute
  • ID selector - targets an element by id
    

* { }                   /* Universal selector */
tag { }                 /* Type selector */
tag::before { }         /* Type selector (pseudo element) */
.class { }              /* Class selector */
.class:hover { }        /* Class selector (pseudo class) */
[attr] { }              /* Attribute selector */
#id { }                 /* ID selector */

Pseudo elements have two colons (:: or ::before) and pseudo classes have one colon (: or :hover).

CSS Specificity

All things being equal, the order in which you use two selectors in the stylesheet matters. So if we have a stylesheet with two classes:

    

.foo {
    color: blue;
}

.bar {
    color: red;
}

Both of these classes are then on the same element:

    

<div class="foo bar">This is an element</div>

The selector closest to the bottom of the stylesheet will win. So in this example, the element's colour will be red because .bar appears closest to the bottom of the stylesheet.
