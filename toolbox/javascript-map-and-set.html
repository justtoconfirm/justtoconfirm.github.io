JavaScript Map & Set

In JavaScript, developers often spend a lot of time deciding on the correct data structure to use. The correct data structure can make it easier to manipulate data later on, saving time and making the code easier to understand. The two main data structures used for storing collections of data are Objects and Arrays (also a type of object).

Objects are used to store key/value pairs, Arrays are used to store indexed lists. Since ES6 was introduced, two new iterable objects have been added to JavaScript: Maps and Sets.

  • Maps - ordered collections of key/value pairs
  • Sets - collections of unique values

Introduction to Maps

A Map is a collection of key/value pairs that can use any data type as a key and can maintain the order of its entries. Maps have elements of both Objects (unique key/value pair collection) and Arrays (an ordered collection), but are more similar to Objects in concept.

Maps can be initialized with the new keyword to provide an empty Map:

    

const map = new Map();

Values can be added to a map with the set() method. The first argument will be the key and the second argument is the value. The following example adds three key/value pairs to our map.

    

map.set('firstName', 'Luke');
map.set('lastName', 'Skywalker');
map.set('occupation', 'Jedi');

With values added, we can begin to see how Maps have elements of both Objects and Arrays. Like an Array, we have a zero-indexed collection and we can see how many items are in the Map by default. Maps use the => syntax to signify key/value pairs as key => value.

A Map does look similar to a regular object but we can use any data type as a key with Map.

    

Map(3)
0: {'firstName' => 'Luke'}
1: {'lastName' => 'Skywalker'}
2: {'occupation' => 'Jedi'}

Getting & Deleting Items from a Map

A Map structure has many built-in properties that make working with their elements much more direct and simple over Objects. We can initialize a new Map to demonstrate the following methods and properties of: delete() and get().

    

const map = new Map([
    ['animal', 'otter'],
    ['shape', 'triange'],
    ['city', 'New York'],
    ['country', 'Bulgaria'],
])

We can use the get() method to retrieve a value by key.

    

map.get('animal');      // "otter"

Should we want to remove a value from the map, we can use the delete() method. This removes an item by key and will return a Boolean of true of an item existed and was deleted. A value of false is returned if it did not match any item.

    

map.delete('city');      // true

One useful benefit of using Maps over Objects is that you can find the size of the Map at anytime, just like an Array. You can use the size property to get the njmber of values in a Map.

    

map.size;       // 4

Should you want to clear a Map of all values, the clear() method can be used.

    

map.clear();
