JavaScript Objects

Objects group together a set of variables and functions to create a model. In an object, variables and functions take on new names. In an object, variables become known as properties and functions become known as methods.

Built-in Objects

Browsers come with a set of built-in objects that represent things like the browser window and the current web page shown in that window. These built-in objects act like a toolkit for creating interactive web pages.

The three sets of built-in objects each offer a different range of tools that help you to write scripts for web pages:

  • Browser Object Model (BOM) - The Browser Object Model contains the objects that represent the current browser window or tab. It contains objects that model things like browser history and the device's screen.
  • Document Object Model (DOM) - The Document Object Model uses objects to create a representation of the current page. It creates a new object for each element (and each individual section of text) within the page. The DOM is used to access and update the contents of a web page.
  • Global JavaScript Objects - The global JavaScript objects represent things that the JavaScript language needs to create a model. For example, there is an object that deals only with dates and times.

As soon as a web page has loaded into your browser, these objects are available for use in your script.

Just like any other object, dot notation can be used to access their properties and methods.

The objects you create will usually be specifically written for your needs and contain functionality, or model the data needed by your script. Built-in functions contain functionality commonly needed by many scripts.

An object model is a group of objects, each of which represent related things from the real world. Together they form a model of something larger.

Browser Object Model

The Browser Object Model (BOM) creates a model of the browser tab or window. The topmost object is the window object, which represents the current browser window or tab. Its child objects represent other browser features.

    

Window
  |- Document
  |- History
  |- Location
  |- Navigator
  |- Screen
  • Window - The current browser window or tab
  • Document - The current web page
  • History - The pages in the browser history
  • Location - URL of the current page
  • Navigator - Information about the browser
  • Screen - The device's display information

The window object's print() method will cause the browser's print dialog box to be shown.

    

window.print();

The screen object's width property will let you find the width of the device's screen in pixels.

    

window.screen.width;

Document Object Model (DOM)

The Document Object Model (DOM) creates a model of the current web page. The topmost object is the document object, which represents the page as a whole. Its child objects represent other items on the page include: <html>, <body>, <title>, <div>, <p>, and text.

The document object's getElementById() method gets an element by the value of its id attribute.

    

document.getElementById('one');

The document object's lastModified property will tell you the date that the page was last updated.

    

document.lastModified;

Global JavaScript Objects

The global objects do not form a single model, they are a group of individual objects that relate to different parts of the JavaScript language. The names of global objects usually start with an uppercase letter.

The String object's toUpperCase() method makes all letters in a variable uppercase.

    

hotel.toUpperCase();

The Math object's PI property will return the value of Pi.

    

Math.PI();

Date & Time

In order to work with dates, you create an instance of the Date object. You can then specify the time and date that you want to represent.

The Date() object constructor tells the JavaScript interpreter that this variable is a date and this in turn allows you to use the Date object's methods to set and retrieve dates and times from this object.

To create a Date object, use the Date() object constructor. The syntax is the same for creating any object with a constructor function and it can be used to create more than one Date object. This is very similar to how you use the Object() constructor to create a new object.

    

var today = new Date();

Here you can see we are using the new keyword and an assignment operator (=) to assign the date and time to a variable.

By default, the Date object will hold today's date and the current time. In JavaScript, dates are stored as a number, the milliseconds since midnight on January 1, 1970 (Unix time/epoch) and is determined by the computer's clock of the user. If you want it to store another date, you must explicitly specify the date and time you want it to hold.

Once the new Date object is created, the date and time will be contained when the JavaScript interpreter encountered the line of code. Once this instance is created, you can use the properties and methods of the Date object.

Here we will use the getFullYear() method of the Date object to return the year of the date being stored in the Date object.

    

var today = new Date();
var year = today.getFullYear();

Creating Objects

To create an empty object literal, use:

    

var hotel = {}

Properties tell us about the object. Methods represent tasks that are associated with the object.

This object represents a hotel. It has five properties and one method. The object is in curly braces and is stored in a variable called hotel.

    

var hotel = {
    name: 'Quay',
    rooms: 40,
    booked: 25,
    gym: true,
    roomTypes: ['twin', 'double', 'suite'],

    checkAvailability: function() {
        return this.rooms - this.booked;
    }
};

This is an example of literal notation. This is the easiest and most popular way to create objects in JavaScript.

The object is stored in a variable called hotel, so you would refer to it as the hotel object.

Like variables and functions, properties and methods have a name and a value. In an object, the name is called a key. Each key and value is separated with a comma. An object cannot have two keys with the same name. The value of a property can be a string, number, Boolean, array, or another object. The value of a method is always a function.

In the checkAvailable() method, the this keyword is used to indicate that it is using the rooms and booked properties of this object.

The function keyword can also be omitted when assigning methods to an object.

    

var hotel = {
    name: 'Quay',
    rooms: 40,
    booked: 25,
    gym: true,
    roomTypes: ['twin', 'double', 'suite'],

    checkAvailability() {
        return this.rooms - this.booked;
    }
};

Constructor Notation

The new keyword and object constructor can be used to create a new blank object. You can then add properties and methods to this object. The Object() constructor is a function that's part of the JavaScript language and is used to create objects. Once an object has been created, dot notation can be used to add the properties and methods.

    

// Create a new blank object
var hotel = new Object();

// Add properties and methods to this object
hotel.name = 'Quay';
hotel.rooms = 40;
hotel.booked = 25;

hotel.checkAvailability = function() {
    return this.rooms - this.booked;  
};

Sometimes you may need many objects to represent similar things. Object constructors can use a function as a template fo creating objects. In this example, we are using a function to create a template with the object's methods and properties.

    

function Hotel(name, rooms, booked) {
    
    this.name = name;
    this.rooms = rooms;
    this.booked = booked;

    this.checkAvailability = function() {
        return this.rooms - this.booked;
    };

}

Here a function called Hotel will be used as a template for creating new objects that represent hotels. Like all functions, it contains statements. In this case, the statements add properties and methods to the object.

This function has three parameters, each sets the value of a property in the object. The methods will be the same for each object created using this template (function).

The this keyword is used instead of the object name to indicate that the property or method belongs to thee object that this function creates.

The name of the function usually begins with an uppercase letter to help remind developers to use the new keyword when they create an object using this template function.

Instances of the Template

You create instances of the object using the constructor function. The new keyword followed by the call to the function creates a new object using the template. The properties for each object you create are given as arguments to be passed into the function parameters.

Here, two objects are created to represent two hotels. Each object needs a different name. When the new keyword calls the constructor function, it creates a new object. Each time it is calld, the arguments passed into the function are different because they are properties of two different hotels. Both objects automatically get the same methods defined within the function (the checkAvailability() method).

    

var quayHotel = new Hotel('Quay', 40, 25);
var parkHotel = new Hotel('Park', 120, 77);

This technique maybe used if your script contains a complex object that needs to be available but might not be used. The object is defined in the function, but only created if it's needed.

Even when many objects are created using the same constructor function, the methods stay the same because they access, update, or perform a calculation based on the data stored in the properties.

Accessing an Object

You can access the properties or methods of an object using dot notation. Square brackets can also be used.

To acceess a property or method of an object, use the name of the object followed by a period (member operator), then the name of the property of method you want to access.

Here are two variables created to hold the name of the hotel and the number of vacant rooms:

    

var hotelName = hotel.name;
var roomsFree = hotel.checkAvailability();

You can also access the properties of an object using square brackets, but not methods. This time the object name is followed by square brackets and the property name is inside.

    

var hotelName = hotel['name'];

Updating an Object

To update the value of a property inside an object, use dot notation once again but using the assignment operator (=) to assign a new value. The same method to get a value is used to set a new value. Here, the name of the hotel is updates from 'Quay' to 'Park':

    

hotel.name = 'Park';

If the property you are attempting to update does not exist, it will be added as a new property to the object.

To delete a property from an object, use the delete keyword followed by the object name and property name.

    

delete hotel.name;

If you want to clear a value of a property, set the value to be a blank string.

    

hotel.name = '';
