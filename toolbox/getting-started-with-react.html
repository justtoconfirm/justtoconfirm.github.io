Getting Started with React

What is React?

  • React is an open-source JavaScript library created by Facebook.
  • Used to build front end user interfaces (UI).
  • React is the view layer of an MVC (Model View Controller) application.

React allows components to be created, which are similar to reusable HTML elements, to quickly build user interfaces. Data within React is stored and handled using state and props.

React Developer Tools

There is a browser extension called React Developer Tools that will make working with React much easier. Download React DevTools for Chrome, or for the browser you prefer to use.

Once installed, when you open DevTools, you'll see a tab for React. This will allow you to inspect components as they're written. You can still go to the Elements tab to see the actual DOM (Document Object Model) output.

Install React

Node Package Manager (NPM) can be used to install React. Begin by creating a new directory where the React application will be built and setup a package.json file. Open your terminal and run the following commands:

        

$ mkdir react-application
$ cd react-application

$ npm init -y

We now want to install the following packages:

  • react - the top-level API of the React library.
  • react-dom - DOM specific methods used by React.

React and ReactDOM were split into two packages from version 0.14. ReactDOM is where you will find the render method.

Once again, run the following commands in your terminal to install react and react-dom as dependencies:

        

$ npm install react react-dom --save

At this stage you may need to setup Webpack and Babel to get React working.
