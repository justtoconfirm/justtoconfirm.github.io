Modules

JavaScript modules, also known as ES modules or ECMAScript modules, were created to help make JavaScript code more organised and maintainable.

What are Modules?

ES modules are the official module system used in JavaScript. There are other module systems that can be used in JavaScript as well. A module in JavaScript is just a file of code. It is reusable and independent.

JavaScript modules, also known as ES modules (ESM) or ECMAScript modules, were created to help make JavaScript code more organised and maintainable.

Modules are te building blocks of a codebase. As an application gets bigger, we can split our code up into multiple files. These are modules. This makes the code more manageable.

When we want to make a module available in other files besides the one it lives in, we can use the export keyword.

React components are often written as reusable modules as the components are reusable within a React application.

Using Modules in HTML

Inside of our HTML file, we can modify the <script> tag to include the type attribute. This will allow us to start using ES modules in our code.

	

<script type="module" src="script.js"></script>
