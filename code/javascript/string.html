String

Strings stores a series of characters such as "First name".

Introduction

Primitive values, like "Dan", cannot have properties and methods because they're not objects. JavaScript treats primitive values as objects when executing methods and properties.

A string can be any text inside double or single quotes.

Length

The length property returns the length of a string:

	

let txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
let length = txt.length;

The slice() method

The slice() method extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters, the start position, and the end position (not included).

JavaScript counts positions from 0, so the first position is 0 and the second position is 1.

	

let string = "Apple, Banana, Kiwi";
let part = string.slice(7, 13);			// "Banana"

If the parameter is negative, the position is counted from the end of the string.
