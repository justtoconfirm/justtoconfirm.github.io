Array

An array is an object which can hold more than one value.

Introduction

In JavaScript, the array data type consists of a list of elements. There are many useful built-in methods avalable for JavaScript developers to work with when using arrays. Methods that modify the original array are known as mutator methods, and methods that return a new value or representation are known as accessor methods.

There is a third class of array methods, known as iteration methods which are methods that operate on every item in an array, one at a time. These methods are closely associated with loops.

Here's an example of a simple array in JavaScript with a list of elements:

	

const cars = ["BMW", "Audi", "Volkswagen"];

The forEach() method calls a function for each element in an array. We can use forEach() to print each item in the cars array to the console:

	

cars.forEach(individualCar => {
	console.log(individualCar);
});

Another way to do this is using the for loop keyword and testing it against the length property of the array:

	

for (let i = 0; i < cars.length; i++) {
	console.log(cars[i]);
}

This will provide the same output in the console as the forEach() method.

The map() method

The map() method creates a new array. The original array is unchanged. We can use map() to change the values of each item in the array without changing the original array. To demonstrate this, we can add something (the word 'car') to the end of each item in the cars array.

	

let addToCars = cars.map(individualCar => {
	return `${individualCar} car`;
});

// Call the function
addToCars;

The filter() method

The filter() method creates a new array with the elements that pass the result of a given test. We could use filter() to return a new array containing only the items in a list that start with a specific letter.

	

// Filter all cars that start with "A" into a new list
let filterList = cars.filter(car => {
	return car[0] === "A";
}); 

// Call the function
filteredList;

filter() is an iteration method and does not mutate the original array.

The reduce() method

The reduce() method will reduce the array to a single value. This is seen commonly with numbers, such as finding the sum of all numbers in an array.

	

let numbers = [42, 23, 16, 15, 4, 8];

// Get the sum of all numerical values
let sum = numbers.reduce((a, b) => {
	return a + b;
});

// Call the function
sum;

The reduce() method is an iteration method that does not mutate the original array.

The find() & findIndex() methods

The find() method returns the first value in an array that passes a given test. With the cars array, we can use find() to test if any of the cars in the array match Volkswagen group vehicles.

	

const isVolkswagenGroup = volkswagenGroup => {
	return ["Audi", "Volkswagen"].includes(volkswagenGroup);
}

cars.find(isVolkswagenGroup);

Since "Audi" was the first entry in the array to satisfy the test in the function, it is the first value to be returned. The find() method can help us to work with arrays that contain many values.

The findIndex() method returns the first index in an array that passes a given test.

	

const isVolkswagenGroup = volkswagenGroup => {
	return ["Audi", "Volkswagen"].includes(volkswagenGroup);
}

cars.findIndex(isVolkswagenGroup);

"Audi" is the first item to match the test and has an index of 1, therefore it is the index number that is returned. If the test is not satisfied, findIndex() will return -1.
