Introduction to Events

Events in JavaScript are actions that have taken place in the browser. Here are a few common events that can happen on a website:

  • The page finishes loading
  • The user clicks a button
  • The user hovers over a dropdown
  • The user submits a form
  • The user presses a key on their keyboard

Adding Events to Buttons

No actions or events are applied by default to a <button> in HTML. We need to use JavaScript to add an event listener to apply functionality for when the button is clicked.

	

<button class="js-button">Button</button>

Using JavaScript we can use the querySelector() document method to match the first element within the document. If no match is found, null is returned in the console.

We use a class named "js-button" here, known as a "js-hook" so we can attach our JavaScript code to our HTML <button> element.

An event listener can now be added to our <button> element using JavaScript to show an alert:

	

const button = document.querySelector('.js-button');

button.addEventListener('click', () => {

	alert('The button has been clicked');

});

The querySelector() method only selects the first element that matches the given selector. If we have multiple buttons, only the first button will have the event listener applied.

We can use the querySelectorAll() document method for multiple elements.

When using querySelectorAll(), we need to loop over the multiple element to apply our code:

	

const button = document.querySelectorAll('.js-button');

button.forEach(selectedButton => {

	selectedButton.addEventListener('click', () => {

		alert('A specific button has been clicked');

	});

});

Toggle CSS classes

We can use this code to toggle CSS classes. Here we have a number of buttons:

	

<button class="js-button btn--active">Button</button>
<button class="js-button">Button</button>
<button class="js-button">Button</button>
<button class="js-button">Button</button>

Our JavaScript code can be updated to apply the btn--active class to whichever button has been clicked:

	

const button = document.querySelectorAll('.js-button');

button.forEach(selectedButton => {

	selectedButton.addEventListener('click', () => {
		
		// Remove the active class from all buttons
		button.forEach(btn => btn.classList.remove('btn--active'));

		// Add the active class to the button being clicked
		this.classList.add('btn--active');

	});
	
});

Pointers

We can use pointers for mouse and touch events (combined) and additional features for pen events.

The pointerdown event covers click, touch and pen events. Touch on mobile devices and click on desktop (mouse) which we can see within the log inside the console.

We can use this to replace click and touch events.

	

const el = document.querySelector('.element');

el.addEventListener('pointerdown', e => {
	
	console.log(e);

});
