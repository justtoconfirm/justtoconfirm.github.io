Kubernetes

Kubernetes (pronounced "koo-ber-net-ees") is an open-source software for deploying and managing containers at scale.

What is Kubernetes?

Kubernetes is an open source orchestration system for automating the management, placement, scaling and routing of containers. It is a container management system developed on the Google platform. It was first developed by Google and contributed to open source in 2014. It is now maintained by the Cloud Native Computing Foundation.

Modern applications are increasingly built using containers, which are microservices packaged with their dependencies and configurations.

Kubernetes helps to manage containerised applications in virtual and cloud environments. It is a flexible container tool to consistently deliver complex applications running on clusters of hundreds to thousands of individual servers.

Kubernetes helps to control the resource allocation and traffic management for cloud applications and microservices. It also includes:

  • Automated Scheduling
  • Automated rollouts and rollback
  • Horizontal scaling & load balancing

What is a Container?

Today's Internet users never accept downtime, therefore, developers have to find a method to perform maintenance and updates without interrupting their services. A container, which is an isolated environment, includes everything needed for the application to run. It makes it easy for developers to edit and deploy apps. Containers have become the preferred method for packaging, deploying, and updating web apps.

With modern web services, users expect applications to be available 24/7, and developers expect to deploy new versions of those applications several times a day. Containers enable applications to be released and updated without downtime.
