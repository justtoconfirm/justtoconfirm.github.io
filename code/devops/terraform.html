What is Terraform?

An open-source tool that offers pre-written modules to build and manage an infrastructure. Terraform is one of the most popular Infrastructure-as-code (IaC) tools used by DevOps teams to automate infrastructure tasks. It is written in the GO programming language.

Install & Setup

Terraform can be installed on your machine on any Operating System including Windows, MacOS, Linux or more. In MacOS, this uses Homebrew.

Terraform uses an easy to understand language, HCL (HashiCorp Configuration Language).

Terraform supports mutiple providers:

  • Amazon Web Services (AWS)
  • Azure
  • ....and many more

Configuration files are a set of files used to describe infrastructure in Terraform and have the file extensions .tf and .tf.json.

Setup

To setup Terraform:

  • Set up a Cloud Account on any cloud provider (e.g. AWS, Azure).
  • Install Terraform.
  • Add a provider (e.g. AWS, Azure).
  • Write configuration file(s).
