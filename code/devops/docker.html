Docker

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers.

What is Docker?

Put simply, Docker is a way for us to make sure our web application has all the things it needs to be able to run no matter what computer is being used. If the computer in question has the Docker runtime installed, then it can run the app. This is called Containerisation.

Docker containers are little containers that contain and run your code in a portable, isolated way.

A Docker file is what you write that lays out what we want our Docker container to be.

A Docker image is all the code bundled and ready to run whenever we need it. Docker will use this and turn it into a running container when we need it.

Install & Setup

To install Docker, go to the Docker website and select the correct choice for our operating system.
