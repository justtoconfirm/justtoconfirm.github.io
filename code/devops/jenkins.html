Jenkins

Jenkins is a self-contained, open-source, automation server.

What is Jenkins?

Jenkins is a self-contained, open-source, automation server. It is used to automate all sorts of tasks relating to building, testing, delivering, and deploying software. Tools like Jenkins can be used within Continuous Integration (CI)/Deployment (CD).

Jenkins can be installed through native packages, Docker, or as a standalone machine with a Java Runtime Engine (JRE) installed.

Continuous Integration (CI) and Continuous Delivery (CD) are integral parts of DevOps. There are many CI/CD tools on the market, and Jenkins is one example.

It is a Java-based, open-source, CI/CD tool.

Installation & Setup

Jenkins can be installed for many Operating Systems or can be put in a Docker Container.

Jenkins has plugins to help customise the experience with Jenkins.
