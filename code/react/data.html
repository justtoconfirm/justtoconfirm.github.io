Getting API Data into React

React is a JavaScript library for building user interfaces. It's the View layer in MVC (Model-View-Controller).

Assume we want to fetch a list of pictures when a page gets loaded. Firstly, we need an API that provides such information. As a page gets loaded, it will create an AJAX request or network request over to an API. The API takes that request and returns a big list of JSON data. We can then take the required data from the JSON returned and use it in our app.

An API provides interactions between one software and another, but not users. External APIs offer data, services, or both.

To fetch data from a REST API, we perform an AJAX request to a REST API which will return the requested information. This can either be done by creating an XMLHttpRequest (XHR) request or use a helper library to make things easier such as:

  • Fetch API
  • Axios

The fetch() method in JavaScript is used to make a request to the server and load the information into our page. The request can be of any APIs that return the data in JSON or XML format. This method returns a promise.

Here we're creating a component that uses the useEffect hook and fetch() method to get data from the API.

	

import {useEffect} from "react";

const App = () => {

	useEffect(() => {
		fetch('https://jsonplaceholder.typicode.com/todos')
			.then(response => response.json())
			.then(json => console.log(json))
	}, []);

	return (
		<>
			Different ways to fetch data
		</>
	);
} 

export default App;

We can also use the useState hook to store the data in the state of a React component. The useEffect hook will be used to mount the component.

	

import {useState, useEffect} from "react";

const App = () => {

	const [Users, fetchUsers] = useState([])

	const getData = () => {
		fetch('https://jsonplaceholder.typicode.com/users')
			.then((response) => response.join())
			.then((response) => {
				console.log(response)
				fetchUsers(response)
			})
		}
	}

	useEffect(() => {
		getData()
	}, [])

	return (
		<>
			<ul>
				{Users.map((item, i) => {
					return <li key={i}>{item.name}</li>
				})}
			</ul>
		</>
	);
} 

export default App;

Open the application and the data from the API will be returned in the console and rendered within the app.

We can use Axios, a promise-based HTTP client designed for Node and browsers, to easily send asynchronous HTTP request to REST APIs and perform create, read, update and delete operations. It is an open-source tool hosted on GitHub and can be imported into vanilla JavaScript or with a library.

First, install Axios using npm:

bash -- 85x24

	npm install axios

This is how we fetch API data in React using Axios:

	

import {useEffect} from "react";
import axios from "axios";

const App = () => {

	useEffect(() => {
		axios.get("https://jsonplaceholder.typicode.com/todos")
			.then((response) => console.log(response.data));
	}, []);

	return (
		<>
			Different ways to fetch Data
		</>
	);
}

export default App;

We could also use a custom React Hook to fetch data from an API into our React application if we are going to reuse the functionality across our application.
