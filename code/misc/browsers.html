Browsers

A Web Browser, or Browser, is a software application for accessing the World Wide Web.

Browser Differences

Despite a variety of applications, all Browsers are created to render web pages. They achieve this with a rendering engine and many browsers share the same rendering engine.

  • Webkit - Safari / Chrome (iOS)
  • Blink - Chrome / Edge (Chromium) / Opera / Brave / Vivaldi - uses Blink, a Chromium-based fork of Webkit.
  • Gecko - Firefox
  • Trident - Internet Explorer (IE)
  • EdgeHTML - Edge (legacy) - this was an update of Trident used in IE. This version of Edge is no-longer supported.

Checking Browser Support

Adding new features to a web page, website and/or application can break browser support. The caniuse.com website can be used to check browser support across versions.

Internet Explorer (IE)

Internet Explorer (IE) is a browser developed by Microsoft using the Trident rendering engine and originally released on August 16, 1995. IE has since been replaced by Edge as the default web browser for Microsoft Windows. Windows 10 features a version of IE11 for legacy applications.

IE Mode in Edge

Edge (Chromium version) will continue to allow legacy applications and websites built for IE to be used using the Internet Explorer legacy mode ("IE Mode") built-in. This mode uses the IE rendering engine within Edge for this purpose. A small IE logo will appear inside the address bar of Edge when this mode is activated.

Support to End

Microsoft is expected to drop support for IE11 and below in Windows 10/11 shortly.

  • August 17, 2021 - IE11 will no longer be supported by Microsoft for online services such as Office 365, OneDrive, and Outlook.
  • November 30, 2021 - IE11 will no longer be supported by Microsoft in Teams and with any other Microsoft application.
  • June 15, 2022 - IE11 support dropped on Windows 10, users to be redirected to use Edge.
