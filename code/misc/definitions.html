Definitions

Technology Terms, Abbreviations and Terminology explained.

C

  • Code Coverage - testing to determine how much code is being tested. For RSA, our code coverage should be greater than 85.0%.
    Code Coverage = Number of lines of code exercised/Total number of lines of code * 100% (as a Percentage).
  • Code Smell - also known as bad smell, in code, refers to any symptom in the source code of a program that possibly indicates a deeper problem. It is a surface indication that usually corresponds to a deeper problem in the system.
