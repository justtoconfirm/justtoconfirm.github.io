What is Google Analytics (GA)?

Google Analytics (GA) is used as an analytics tool to reveal who is visiting your website. The data collected can then be used to continually improve your website.

A property in Google Analytics (GA) needs to be setup which allows you to generate a tracking ID. This tracking ID is a unique identifier created for your Google Analytics (GA) account.

Google Analytics allows:

  • Tracking the number of visitors to our website.
  • Understand the time spent on the website.
  • Record goal conversions.
  • much more...

Google Analytics (GA) will automatically begin measuring traffic to your website and user interactions such as: page scrolls, site searches, engagement, downloads, and much more.

The Google Tag Assistant Chrome extension can be used to test that GA has been setup on a website. This is a legacy product but still works.

gtag.js

Once the tracking ID has been identified we will ned to get our tracking code snippet which can be found as the Global Site Tag (gtag.js). The gtag.js is the tracking code for this property.

gtag.js was launched in 2017 and is a newer and more expanded version of analytics.js.

This gtag.js code snippet will need to be copied and pasted into every webpage we want to track in Google Analytics (GA) within our website.

This gtag.js snippet is often pasted right after the opening <head> tag on each page of the website you want to track.

If using Google Tag Manager (GTM), we can use a tag to add Google Analytics (GA) instead of using the gtag.js code snippet.

Introducing Google Analytics 4 (GA4)

Google Analytics 4 (GA4) is a web analytics tool from Google that lets you track and measure user traffic and behaviour on your website. It is used by an estimated 28.8 million websites, giving an estimated 85.9% share of the web analytics market.

GA4 replaced Universal Analytics in October 2020 and is now the default Google Analytics (GA) property.

Data within Google Analytics 4 (GA4) can be exported into a PDF or CSV file format.

Page View Tracking

A pageview is defined as a view of a page on your website that is being tracked by the Analytics tracking code.

If a user clicks reload after reaching a web page, this is counted as an additional pageview. if a user navigates to a different page than returns to the original page, a second pageview is recorded.

To confirm if pageview is defined:

  • Open the Network tab in the browser DevTools.
  • Refresh the webpage.
  • Type 'google' into the Filter field inside the Network tab.
  • t=pageview within the Name field would suggest that GA pageview is active. We may see this as a URL parameter (t is the track type).
