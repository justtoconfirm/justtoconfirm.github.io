Node

Node is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser.

Package Manager

A package manager is a tool developers use to automate finding, downloading, installing, configuring and updating packages.

Examples of package managers include npm (Node Package Manager) and Yarn.

npm gets installed automatically while installing Node.

A package registry is a database (storage) for thousands of packages (libraries, plugins, frameworks, and/or tools). In other words, a package registry is the place packages get published to and installed from.

The npm registry and GitHub packages are two popular package registries.

It is best to install Yarn through npm. So first, install npm using Node. Once npm has been installed, proceed to install Yarn globally like so:

bash -- 85x24

	npm install yarn -g

To check if npm or Yarn has been installed:

bash -- 85x24

	npm -v
	yarn -v

If we need to upgrade to the latest npm version:

bash -- 85x24

	npm install npm@latest -g

To list all npm package that we have installed globally on our system, we can use the following command:

bash -- 85x24

	npm list -g
